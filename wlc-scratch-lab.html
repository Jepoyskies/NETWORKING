🏗️ The "From Scratch" Blueprint

How to build, cable, and configure an Enterprise WLC network on a blank Packet Tracer canvas.

1
The Hardware (What to Drag)

Go to the bottom-left corner of Packet Tracer and drag these exact models onto the white canvas:

📡 Wireless Controller

Wireless Devices -> 3504

🛸 Access Point

Wireless Devices -> LAP-PT or 3702i

🎛️ Network Switch

Switches -> 2960

🖥️ RADIUS Server

End Devices -> Server-PT

💻 Admin PC

End Devices -> PC-PT

📱 Wireless Client

End Devices -> Laptop-PT

2
Cabling & Power

Connect everything using Straight-Through Copper cables (the solid black line). Connect the Server, Admin PC, WLC, and AP all directly into the 2960 Switch.

🚨 DEADLY PACKET TRACER TRAP #1: Powering the AP

Unlike real life where switches use PoE, in Packet Tracer, the LAP-PT is completely dead by default! You must click the AP, go to the Physical tab, find the black power adapter cord in the corner, and drag it into the AP's power port. If you skip this, the AP will never turn on and you fail the lab.

🚨 DEADLY PACKET TRACER TRAP #2: The Laptop NIC

The Laptop-PT comes with an Ethernet port by default. Click the laptop, turn the physical power button OFF, drag the Ethernet port into the trash, and drag the WPC300N (Wireless Card) into the empty slot. Turn it back ON.

3
The Underlying Network

Before the WLC can work, the devices need IP addresses so they can talk to each other. Here is the standard config you must type into the devices to set up the foundation.

1. Configure the RADIUS Server: Click the Server -> Desktop -> IP Configuration. Set IP to 192.168.1.253, Subnet 255.255.255.0. Then go to Services -> AAA. Turn it ON. Add Client Name WLC-1, IP 192.168.1.254, Secret RadiusPW, Type Radius.

2. Configure the Switch (DHCP for the AP): The AP needs an IP address to find the WLC. Type this into the 2960 Switch CLI:

enable
configure terminal
ip dhcp pool AP-POOL
 network 192.168.1.0 255.255.255.0
 default-router 192.168.1.1
 exit
interface vlan 1
 ip address 192.168.1.2 255.255.255.0
 no shutdown

3. Configure the Admin PC: Click the Admin PC -> Desktop -> IP Configuration. Set it statically to 192.168.1.10 so you can log into the WLC.

4
Practice the WLC Configuration GUI

Now, open the Admin PC Web Browser and type https://192.168.1.254 to configure the WLC. Practice typing the exact required settings in the mock GUI below.
