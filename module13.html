Module 13: WLAN Configuration

Goal: Learn how to configure a Home Router and an Enterprise WLC.

13.1 Remote Site WLAN Configuration

Home Routers

A wireless home router is actually 3 things in one: an Access Point, a Switch, and a Router! When setting it up, your first step must always be to change the default administrative password.

It uses NAT to translate your private IPs to one public IP. You can also configure QoS to prioritize VoIP traffic, and Port Forwarding to let outside internet traffic reach a specific device inside your LAN.

13.2 Configure a Basic WLAN on the WLC

WLC GUI & Network Summary

To configure a WLC, you log into its graphical web interface (GUI). The first page you see is the Network Summary, a dashboard showing active APs, clients, and rogue devices. Clicking the "Advanced" button gives you full control.

5 Steps to Configure a Basic WLAN:

  1. Create the WLAN: Click WLANs > Go. Give it a Profile Name and an SSID.
  2. Enable it: Check the "Enabled" box.
  3. Select Interface: Map the WLAN to a specific VLAN interface (e.g., VLAN 5).
  4. Secure it: Go to the Security Tab -> Layer 2 -> select WPA2 and enter your Pre-Shared Key (PSK).
  5. Verify: Check the WLANs menu to ensure it is active.

13.3 Configure a WPA2 Enterprise WLAN

Enterprise Setup (RADIUS & SNMP)

Enterprise networks don't use shared passwords. They use a RADIUS Server. To configure this:

  1. Click the SECURITY tab > RADIUS > Authentication > New.
  2. Enter the IP address of the RADIUS server and the "Shared Secret" key.

To monitor the network for errors, you can configure the WLC to send log messages (Traps) to an SNMP server via the MANAGEMENT tab.

*Note: Each new WLAN on the WLC needs its own Virtual Interface mapped to a VLAN, and its own DHCP scope!

13.4 Troubleshoot WLAN Issues

Troubleshooting Methodology

If a wireless client can't connect, you should check the PC's ipconfig, try to ping the gateway, check if the security settings match, and verify physical placement (are they too far away?).

If the network is just slow:

  • Split the Traffic: The 2.4 GHz band is crowded. The easiest way to fix a slow network is to name the 2.4 GHz and 5 GHz networks differently, and force users streaming multimedia to connect to the 5 GHz band.
  • Update Firmware: Manufacturers release updates to fix bugs. You can push a firmware upgrade to all APs at once from the WLC's Wireless > Access Points menu.
