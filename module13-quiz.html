1. A user is configuring a wireless access point and wants to prevent any neighbors from discovering the network. What action does the user need to take?
Correct Answer: Disable SSID broadcast.

Deep Dive: To stop neighbors from seeing your Wi-Fi name on their phones, you must turn off the Beacon broadcasts. This is known as SSID Cloaking. This forces the AP into Active Mode, meaning legitimate users must manually type the hidden name to connect.

  • A. Enable WPA encryption: (Incorrect) This adds a password, but the network name is still publicly visible.
  • B. Configure DMZ: (Incorrect) DMZ exposes a local device to the public internet, completely unrelated to Wi-Fi visibility.
  • C. Configure DNS: (Incorrect) DNS resolves domain names to IP addresses.
  • D. Disable SSID broadcast: (Correct) Prevents the AP from announcing the network name to casual observers.
2. When a wireless network in a small office is being set up, which type of IP addressing is typically used on the networked devices?
Correct Answer: Private

Deep Dive: Internal office networks always use Private IP addresses (like 192.168.1.x or 10.0.0.x) provided by a DHCP server. These addresses are free, abundant, and easily managed. Because they cannot cross the public internet, the office router uses NAT to translate them when users browse the web.

  • A & C. Network / Wireless: (Incorrect) These are not standard classifications for IP address types.
  • B. Private: (Correct) The standard, best practice for internal LAN and WLAN device addressing.
  • D. Public: (Incorrect) Public IPs cost money and are strictly regulated. You would not assign them to internal office laptops.
3. A user has just purchased a generic home router and would like to secure it. What should be done to help secure the wireless home router?
Correct Answer: Change the default administrator password.

Deep Dive: Fresh out of the box, most home routers have globally known login credentials (like admin/admin or admin/password). If you don't change this immediately, anyone connected to your network can log into the router's web interface and lock you out. It is the absolute first step in securing a new device.

  • A. Change the default SSID: (Incorrect) While a good idea, changing the SSID does not actually secure the configuration portal of the router itself.
  • B. Allow only IPv6: (Incorrect) Many older devices still rely on IPv4, this would break connectivity.
  • C. Set a private IPv4 network: (Incorrect) Routers do this by default automatically.
  • D. Change the default administrator password: (Correct) Secures the device's management interface from unauthorized tampering.
4. Which protocol could be used by a company to monitor devices such as a wireless LAN controller (WLC)?
Correct Answer: SNMP

Deep Dive: Network administrators use SNMP (Simple Network Management Protocol) to collect health metrics, bandwidth usage, and error logs from devices like a WLC. The devices send "Traps" to the SNMP server whenever an issue arises.

  • A. SNMP: (Correct) The industry-standard protocol for monitoring and managing network infrastructure.
  • B. SSH: (Incorrect) SSH provides encrypted command-line access, but it is not an automated monitoring tool.
  • C. NTP: (Incorrect) NTP keeps the system clocks perfectly synchronized.
  • D. PAT: (Incorrect) Port Address Translation shares a single public IP among many private IPs.
5. When configuring a Cisco 3500 series wireless LAN controller (WLC) for a WPA2 Enterprise WLAN, what has to be created on the WLC before creating the new WLAN?
Correct Answer: A VLAN for the wireless network

Deep Dive: A WLC sits between the wireless airwaves and the wired physical network. To pass traffic properly, every new WLAN you create must be mapped to a specific logical network on the switch side. Therefore, you must first create a dynamic virtual interface and assign it a VLAN ID before the WLAN can be established.

  • A & C. Security Policy/Module: (Incorrect) Security settings like WPA2 are configured *during* the creation of the WLAN profile itself.
  • B. A new SSID: (Incorrect) The SSID is created concurrently with the WLAN.
  • D. A VLAN for the wireless network: (Correct) The foundation that links the wireless SSID to the wired network.
6. What is a DHCP scope as it relates to a WLAN configured on the WLC controller?
Correct Answer: A pool of IP addresses for WLAN clients

Deep Dive: If you configure a WLC to hand out IP addresses internally, you must define a Scope. In standard DHCP terminology, a "Scope" is simply the valid range (pool) of IP addresses available to be leased out to laptops and phones connecting to the network.

  • A. Security rules: (Incorrect) That would refer to DHCP Snooping, not a Scope.
  • B. Distance allotted: (Incorrect) DHCP doesn't govern physical distance.
  • C. A pool of IP addresses: (Correct) Includes the range, the subnet mask, default gateway, and DNS server.
  • D. A corporate plan: (Incorrect) While planning is good, a Scope is a specific technical configuration on the server.
7. Why would a technician configure a passphrase for a WLAN on a wireless router?
Correct Answer: To configure wireless client authentication

Deep Dive: When you set a Wi-Fi password (technically called a Pre-Shared Key or Passphrase) using WPA2 or WPA3, you are establishing Authentication. It forces any client trying to access the network to prove they are authorized to be there by providing the correct password.

  • A. Client authentication: (Correct) Verifies the identity/authorization of users trying to join the WLAN.
  • B. Protect from cabling directly: (Incorrect) A Wi-Fi passphrase does not affect physical ethernet ports.
  • C & D. Protect SSID/Configuration: (Incorrect) You protect the router's settings by setting a strong Admin password, not a Wi-Fi passphrase.
8. A customer installs a wireless access point in a closet next to the kitchen. Wireless performance degrades heavily when the cordless phone or microwave oven is in use. What is the possible reason?
Correct Answer: The wireless signal is in the same radio frequency range as the household devices are in.

Deep Dive: The standard 2.4 GHz frequency band used by Wi-Fi is unregulated (ISM band). Unfortunately, traditional microwave ovens and old cordless phones operate on the exact same 2.4 GHz radio frequency! When turned on, they blast "noise" into the airwaves, causing massive accidental interference and dropping Wi-Fi packets.

  • A. Cordless phone joins WLAN: (Incorrect) Dumb appliances don't join your Wi-Fi network; they just emit disruptive radio waves.
  • B. Close to walls: (Incorrect) While walls degrade signal, they don't explain why it gets worse *only* when the microwave is turned on.
  • C. Same radio frequency range: (Correct) 2.4 GHz interference is a classic Wi-Fi troubleshooting issue.
  • D. Surge of electricity: (Incorrect) Power draw does not degrade radio signal quality directly.
9. What functionality is required on routers to provide remote workers with VoIP and videoconferencing capabilities?
Correct Answer: QoS

Deep Dive: Voice (VoIP) and video are highly sensitive to latency. If someone starts downloading a huge file, a Zoom call will instantly stutter and freeze. Quality of Service (QoS) allows the router to prioritize real-time voice and video packets, sending them out first and making file downloads wait in line.

  • A & B. IPsec / VPN: (Incorrect) These provide encrypted tunneling for remote workers, but they don't ensure the call doesn't lag.
  • C. PPPoE: (Incorrect) An outdated DSL internet connection protocol.
  • D. QoS: (Correct) Prioritizes critical, time-sensitive traffic over standard data.
10. A wireless router is displaying the IP address of 192.168.0.1. What could this mean?
Correct Answer: The wireless router still has the factory default IP address.

Deep Dive: Almost all generic SOHO routers (Linksys, TP-Link, Netgear) ship from the factory with their LAN interface statically set to 192.168.0.1 or 192.168.1.1. Seeing this address highly suggests the router is fresh out of the box and has not been customized for a specific enterprise network.

  • A. Factory default IP: (Correct) 192.168.0.1 is the most common default default gateway address used by manufacturers.
  • B. Frequencies on channel 1: (Incorrect) IP addressing has absolutely nothing to do with radio frequency channels.
  • C. NAT is not working: (Incorrect) A local IP address does not indicate a NAT failure.
  • D. Dynamic allocation functioning: (Incorrect) The router's *own* address is usually static, it's the *clients* that get dynamic addresses.
11. A laptop cannot connect to a wireless access point. Which two troubleshooting steps should be taken first? (Combined)
Correct Answer: Ensure that the wireless NIC is enabled AND Ensure that the wireless SSID is chosen.

Deep Dive: When troubleshooting, always start with the absolute simplest Layer 1 / Layer 2 problems (the "Is it plugged in?" approach). Most laptops have an "Airplane Mode" button that kills the Wi-Fi card. Ensure the card is actually turned on, and ensure the user clicked the correct SSID (Network Name) from their list.

  • A, B, C. Antennas / Frequencies / Media: (Incorrect) Laptops have built-in antennas that cannot be detached. The NIC handles frequency selection automatically. "Network media" applies more to switching physical cables.
  • D. NIC enabled & SSID chosen: (Correct) The most logical and common first steps for end-user Wi-Fi troubleshooting.
12. In setting up a small office network, the administrator decides to assign private IP addresses dynamically. Which feature must be enabled on the company router in order for office devices to access the internet?
Correct Answer: NAT

Deep Dive: Private IP addresses (like 10.x.x.x) are permanently banned from crossing the global internet. To allow the office devices to browse the web, the router uses Network Address Translation (NAT) to swap out their private IPs for the router's one valid Public IP before sending packets to the ISP.

  • A. MAC filtering: (Incorrect) An access-control security feature.
  • B. UPnP: (Incorrect) Allows devices to automatically open firewall ports (often a security risk).
  • C. QoS: (Incorrect) Prioritizes important traffic.
  • D. NAT: (Correct) The essential mechanism that bridges Private networks to Public networks.
13. What is a difference between autonomous APs that operate in a home environment and controller-based APs that operate in a corporate environment?
Correct Answer: Controller-based APs can be automatically configured and managed by a WLAN controller.

Deep Dive: An Autonomous AP has its own brain and requires manual setup. A Controller-based (Lightweight) AP is "dumb". When plugged in, it searches for a WLC using CAPWAP. The WLC then pushes all the settings, passwords, and firmware automatically, allowing a single admin to manage thousands of APs simultaneously.

  • A. Require initial config: (Incorrect) Lightweight APs do *not* require manual configuration; they pull it from the WLC.
  • B. Do not support PoE: (Incorrect) Both types can run on Power over Ethernet.
  • C. Automatically configured by WLC: (Correct) The primary advantage of enterprise Controller-based APs.
  • D. Incorporate functions of router/switch: (Incorrect) While SOHO routers do this, dedicated enterprise Autonomous APs do not necessarily act as routers/switches.
14. Which WLC tab would a network administrator typically use to see a summary view of the most heavily used WLANs including the number of clients using a particular WLAN?
Correct Answer: Monitor

Deep Dive: On a Cisco WLC, the Monitor tab acts as your main dashboard. It provides a visual summary showing system uptime, total active clients, rogue APs detected, and graphs displaying which WLANs are experiencing the heaviest traffic loads.

  • A. Commands: (Incorrect) Used for rebooting, uploading firmware, or pinging devices.
  • B. Monitor: (Correct) The default landing page providing a bird's-eye view of network health and client statistics.
  • C. Controller: (Incorrect) Used to configure NTP, IPv6, and internal WLC hardware settings.
  • D. WLANs: (Incorrect) Used to *create/edit* the networks and change security passwords, not view traffic graphs.
15. Users on an IEEE 802.11n network are complaining of slow speeds. The network administrator checks the AP and verifies it is operating properly. What can be done to improve the wireless performance in the network?
Correct Answer: Split the wireless traffic between the 802.11n 2.4 GHz band and the 5 GHz band.

Deep Dive: 802.11n routers are dual-band. If everyone connects to the single default SSID, most devices will crowd into the longer-range, highly congested 2.4 GHz band. Renaming the networks (e.g., "Office" and "Office_5G") allows you to push laptops streaming video onto the faster 5 GHz band, clearing the traffic jam.

  • A. Split traffic: (Correct) Moving high-bandwidth users to 5 GHz instantly alleviates 2.4 GHz congestion.
  • B. Switch to 802.11g: (Incorrect) 802.11g is much older and slower (max 54 Mbps). This would ruin performance.
  • C. Mixed mode: (Incorrect) Mixed mode slows the network down by accommodating legacy devices.
  • D. Change authentication: (Incorrect) Passwords do not affect radio frequency transmission speeds.
