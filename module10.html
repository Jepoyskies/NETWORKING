Module 10: LAN Security Concepts

Goal: Learn how to protect Layer 2 (Switches) from hackers.

10.1 Endpoint Security

Network Attacks

Attackers use DDoS, Data Breaches, and Malware (like Ransomware) to compromise endpoints.

Defenses include VPNs, NAC, Cisco ESA for emails (to stop Phishing), and Cisco WSA for web browsing.

10.2 Access Control

AAA & 802.1X

Instead of weak local passwords on the SSH line, professionals use AAA with RADIUS for central authentication. For physical wall-jack security, 802.1X is used to keep the switch port closed until log-in.

10.3 to 10.5 Switch Attacks & Mitigations

Layer 2 Attacks (Memorize These Pairs!)

The Attack The Defense / Mitigation
MAC Address Flooding (using macof) Port Security
VLAN Hopping & Double-Tagging Disable DTP (auto trunking), manage the Native VLAN

ARP Spoofing / Poisoning Dynamic ARP Inspection (DAI)
IP Spoofing IP Source Guard (IPSG)
