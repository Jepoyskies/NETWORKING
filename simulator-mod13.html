📖 How to answer the Packet Tracer Lab (Answer Key)

Use this guide to study the exact inputs required to get 100% on the final Module 13 Capstone.

1. Home Wireless Router

  • Internet Connection: DHCP
  • Router IP: 192.168.6.1 / Subnet: 255.255.255.224
  • DHCP Pool: Start IP 192.168.6.3, Max Users 20
  • Static DNS: 10.100.100.252
  • Wireless 2.4GHz: SSID HomeSSID, Channel 6
  • Security: WPA2-Personal, Passphrase Cisco123

2. WLC-1 Management (The Tricky Part)

  • Controller > Interfaces:
    • Create WLAN 2 (VLAN 2, IP 192.168.2.254, GW 192.168.2.1)
    • Create WLAN 5 (VLAN 5, IP 192.168.5.254, GW 192.168.5.1)
  • Controller > Internal DHCP Server:
    • Scope management (Pool: 192.168.100.235 - .245, Net: 192.168.100.0, Def Router: 192.168.100.1)
  • Security Tab:
    • Add RADIUS Server: IP 10.6.0.254, Secret RadiusPW
  • Management Tab:
    • Add SNMP Trap Receiver: Community WLAN, IP 10.6.0.254
  • WLANs Tab (🚨 DO NOT FORGET THESE GOTCHAS):
    • WLAN 2: SSID SSID-2, map to Interface WLAN 2, Security WPA2-PSK (Cisco123).
    • WLAN 5: SSID SSID-5, map to Interface WLAN 5, Security 802.1x (Select your Radius server).
    • For BOTH WLANS: You MUST check Enabled on the General tab.
    • For BOTH WLANS: Go to Advanced Tab -> Scroll down -> Check FlexConnect Local Switching & FlexConnect Local Auth.

3. Connecting the Hosts (PC Wireless App)

  • Wireless Host 1: Connect to SSID-2 using WPA2-Personal and password Cisco123.
  • Wireless Host 2: Connect to SSID-5 using WPA2-Enterprise. Login name userWLAN5, password userW5pass.

Test Network Configurations (Success Indicators)

  1. Configure Home Wireless Router for DHCP and WPA2-PSK.
  2. Configure WLC Interfaces and Management DHCP scope.
  3. Configure WLC with RADIUS, SNMP, and multiple WLANs.
  4. Connect Hosts securely using WPA2-PSK and WPA2-Enterprise.
