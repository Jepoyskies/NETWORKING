Module 12 Quiz: WLAN Concepts

1. In the context of mobile devices, what does the term tethering involve?
Correct Answer: Connecting a mobile device to another mobile device or computer to share a network connection

Deep Dive: Tethering is a process that allows you to turn your smartphone into a portable router. By using your phone's cellular data connection (WWAN), you can "tether" a laptop or PC to it via USB, Bluetooth, or by creating a Wi-Fi hotspot so the laptop can access the Internet.

  • A. hands-free headset: (Incorrect) This is a basic Bluetooth audio pairing, not network sharing.
  • B. share a network connection: (Correct) This perfectly defines tethering.
  • C. 4G cellular network: (Incorrect) This is just standard mobile data access, not tethering.
  • D. USB to charge: (Incorrect) This is charging, not data/network sharing.
2. Which feature of 802.11n wireless access points allows them to transmit data at faster speeds than previous versions of 802.11 Wi-Fi standards did?
Correct Answer: MIMO

Deep Dive: Older standards like 802.11a/b/g used a single antenna. 802.11n introduced MIMO (Multiple-Input Multiple-Output), which uses multiple antennas to transmit multiple data streams simultaneously, drastically increasing bandwidth and speed.

  • A. SPS: (Incorrect) Single Point Setup is a Cisco technology for clustering APs, not a speed enhancement.
  • B. MIMO: (Correct) The antenna technology that skyrocketed Wi-Fi speeds starting with 802.11n.
  • C. MITM: (Incorrect) Man-in-the-Middle is a cyberattack.
  • D. WPS: (Incorrect) Wi-Fi Protected Setup is an insecure push-button method for joining a network.
3. Which method of wireless authentication is currently considered to be the strongest among the given choices?
Correct Answer: WPA2

Deep Dive: Wireless security has evolved over time. WEP was the first, but it is easily hacked. WPA was a temporary fix. WPA2 introduced military-grade AES encryption, making it the strongest on this list (though WPA3 is the modern successor).

  • A. Open: (Incorrect) No password or encryption at all.
  • B. WPA2: (Correct) Uses AES encryption, making it highly secure.
  • C. WEP: (Incorrect) Extremely outdated and vulnerable to cracking in minutes.
  • D. Shared key / WPA: (Incorrect) Older standards using weaker TKIP encryption.
4. Which parameter is commonly used to identify a wireless network name when a home wireless AP is being configured?
Correct Answer: SSID

Deep Dive: The human-readable name of a Wi-Fi network (like "Starbucks_Guest" or "Home_WiFi") is technically called the SSID (Service Set Identifier). It is required for a client to distinguish between multiple nearby networks.

  • A & B. BESS / ESS: (Incorrect) ESS refers to the physical topology of multiple APs connected together, not the text name of the network.
  • C. SSID: (Correct) The 32-character string that identifies a WLAN.
  • D. Ad hoc: (Incorrect) A peer-to-peer network topology with no central AP.
5. Which characteristic describes a wireless client operating in active mode?
Correct Answer: Must know the SSID to connect to an AP

Deep Dive: In Active Mode, the Access Point uses SSID Cloaking (it stops broadcasting its name). Because the network is invisible, the client device cannot passively wait to see it on a list. The client MUST already know the exact SSID so it can actively broadcast a Probe Request searching for it.

  • A & B. Security / Changing channels: (Incorrect) These apply to both active and passive configurations.
  • C. Must know the SSID: (Correct) Active mode requires the client to explicitly ask for the hidden network by name.
  • D. Only listens for beacons: (Incorrect) This describes Passive Mode.
6. Which IEEE standard operates at wireless frequencies in both the 5 GHz and 2.4 GHz ranges?
Correct Answer: 802.11n

Deep Dive: The IEEE 802.11 standards dictate frequency bands. 802.11a is 5 GHz only. 802.11b and g are 2.4 GHz only. 802.11n was the first major standard to be "Dual Band", meaning it can broadcast on both 2.4 GHz and 5 GHz to support older and newer devices.

  • A. 802.11a: (Incorrect) 5 GHz only.
  • B & D. 802.11b/g: (Incorrect) 2.4 GHz only.
  • C. 802.11n: (Correct) Operates in both bands simultaneously.
7. Which statement describes an autonomous access point?
Correct Answer: It is a standalone access point.

Deep Dive: An Autonomous AP has all of its "brains" built into it. You log directly into it to configure its passwords, channels, and security. It operates completely independently. In contrast, "Lightweight" APs have no brains; they require a WLC to push configurations to them.

  • A, B, C. Managed by WLC / Large networks: (Incorrect) These describe Lightweight Access Points (LAPs) running CAPWAP.
  • D. Standalone: (Correct) Autonomous means it governs itself without needing a central controller.
8. Which two roles are typically performed by a wireless router that is used in a home or small business? (Combined)
Correct Answer: Access point AND Ethernet switch

Deep Dive: A typical home "Wi-Fi Router" (like a Netgear or Linksys) is actually three separate devices shoved into one plastic box: 1) A Router to do NAT and connect to the ISP, 2) An Ethernet Switch (the 4 LAN ports on the back), and 3) A Wireless Access Point (the antennas for Wi-Fi).

  • A. Access point & Ethernet switch: (Correct) These are built-in features of SOHO wireless routers.
  • B & D. RADIUS Server: (Incorrect) A RADIUS server is a separate enterprise-grade authentication server.
  • C. WLAN controller: (Incorrect) Small businesses don't use WLCs; they just use the router autonomously.
9. Which protocol and port numbers are used by both IPv4 and IPv6 CAPWAP tunnels? (Combined)
Correct Answer: UDP AND 5246 & 5247

Deep Dive: CAPWAP builds two tunnels between a Lightweight AP and a WLC. It uses UDP port 5246 for the Control messages (encrypted via DTLS), and UDP port 5247 for the actual Data payload (the user's web traffic).

  • A, C, D. Wrong protocols/ports: (Incorrect) CAPWAP does not use TCP, ICMP, or ports 17/163.
  • B. UDP 5246 & 5247: (Correct) The exact transport layer protocol and ports designated for CAPWAP.
10. If three 802.11b access points need to be deployed in close proximity, which three frequency channels should be used? (Combined)
Correct Answer: 1, 6, AND 11

Deep Dive: 802.11b operates in the 2.4 GHz band, which is extremely narrow. Radio signals "bleed" into neighboring channels. To prevent APs from causing static and interference with each other (co-channel interference), you must use the only three channels that do not overlap at all: 1, 6, and 11.

  • A, B, D. Overlapping channels: (Incorrect) Channels like 3, 5, and 8 will bleed into each other, causing massive network lag.
  • C. 1, 6, and 11: (Correct) The standard non-overlapping channel scheme for 2.4 GHz WLANs.
11. Which type of telecommunication technology is used to provide Internet access to vessels at sea?
Correct Answer: Satellite

Deep Dive: If you are in the middle of the Pacific Ocean on a naval vessel or cruise ship, there are no physical cell towers for hundreds of miles. The only way to get a WWAN connection is to beam the signal directly to space using Satellite technology.

  • A & C. Cellular / WiMax: (Incorrect) These require land-based towers and only cover cities or rural land areas.
  • B. Municipal WiFi: (Incorrect) A WMAN that only covers a city.
  • D. Satellite: (Correct) Provides global internet coverage, perfect for ships and airplanes.
12. Which wireless network topology is being configured by a technician who is installing a keyboard, a mouse, and headphones, each of which uses Bluetooth?
Correct Answer: Ad hoc mode

Deep Dive: When devices connect *directly* to each other without using a central router or Access Point, it is called a peer-to-peer or Ad hoc network. Because Bluetooth devices (like mice and headphones) pair directly to the PC, they form an ad hoc WPAN.

  • A. Hotspot: (Incorrect) A hotspot shares internet via a cellular connection.
  • C. Ad hoc mode: (Correct) Independent Basic Service Set (IBSS). Devices speak peer-to-peer.
  • D. Infrastructure mode: (Incorrect) This requires a central AP that all clients connect through.
13. Which type of wireless topology is created when two or more Basic Service Sets are interconnected by Ethernet?
Correct Answer: ESS

Deep Dive: A Basic Service Set (BSS) is simply a single Access Point and its clients. If a building is too large for one AP, you install multiple APs and wire them together through an Ethernet switch. When multiple BSSs are linked together to broadcast the same SSID, it becomes an ESS (Extended Service Set).

  • A & C. IBISS / Ad hoc: (Incorrect) Peer-to-peer networks with no APs.
  • B. ESS: (Correct) Multiple interconnected BSSs forming a larger WLAN.
  • D. BSS: (Incorrect) A single, standalone AP.
14. What Wi-Fi management frame is regularly broadcast by APs to announce their presence?
Correct Answer: Beacon

Deep Dive: In Passive Mode, an Access Point constantly shouts out a management frame multiple times a second saying, "I am here, my SSID is 'GuestNet', and I support WPA2." This specific frame is called a Beacon.

  • A. Probe: (Incorrect) Probe requests are sent by the *client* searching for networks.
  • B. Beacon: (Correct) Regularly broadcast by the AP to advertise its presence.
  • C & D. Association / Authentication: (Incorrect) These frames are only used dynamically when a client is actively trying to log into the AP.
