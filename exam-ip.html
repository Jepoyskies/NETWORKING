Practice Exam: IP Addressing (All Questions)

1. What is the prefix length notation for the subnet mask 255.255.255.224?

224 in binary is 11100000. 8 + 8 + 8 + 3 = 27 bits turned on.

2. How many valid host addresses are available on an IPv4 subnet that is configured with a /26 mask?

A /26 leaves 6 host bits (32 - 26 = 6). Formula is 2^6 - 2 = 62 usable hosts.

3. Which subnet mask would be used if 5 host bits are available?

If there are 5 host bits, there are 27 network bits (/27). A /27 mask is .224.

4. A network administrator subnets the 192.168.10.0/24 network into subnets with /26 masks. How many equal-sized subnets are created?

You borrowed 2 bits (from /24 to /26). 2^2 = 4 subnets.

5. Which host address falls into the 192.168.1.64/27 subnetwork?

The .64/27 subnet ranges from .64 to .95. Only .68 falls in this range.

6. An administrator wants to create four subnetworks from the network address 192.168.1.0/24. What is the network address and mask of the SECOND useable subnet?

To get 4 subnets, borrow 2 bits (/26). The subnets jump by 64. Networks are .0, .64, .128, .192. The second is .64.

7. How many bits must be borrowed from the host portion of an address to accommodate a router with five connected networks?

2^2 = 4 (not enough). 2^3 = 8 (enough for 5 networks). You must borrow 3 bits.

8. How many host addresses are available on the 192.168.10.128/26 network?

A /26 prefix gives 6 host bits. 2^6 = 64. Subtract 2 (network and broadcast) = 62 usable addresses.

9. How many host addresses are available on the network 172.16.128.0 with a subnet mask of 255.255.252.0?

A mask of .252 is a /22. This leaves 10 host bits. 2^10 - 2 = 1022 usable IPs.

10. Which three blocks of addresses are defined by RFC 1918 for private network use?

The standard private IP blocks are 10.x.x.x, 172.16 to 172.31, and 192.168.x.x.

11. What is the broadcast address for network 172.16.16.0/22?

A /22 increments in blocks of 4 in the third octet. The network is 16.0 to 19.255. The broadcast is .19.255.

12. A site administrator has been told that a particular network must accommodate 126 hosts. Which subnet mask would be used?

126 hosts requires 7 host bits (2^7 - 2 = 126). 32 - 7 = /25. A /25 mask is .128.

13. Considering the addresses already used and having to remain within the 10.16.10.0/24 network range, which subnet address could be assigned to a network containing 25 hosts?

25 hosts requires 5 host bits (/27). A /27 provides exactly 30 usable hosts, which perfectly fits 25.

14. Which address prefix range is reserved for IPv4 multicast?

Class D addresses (224 to 239) are reserved for multicasting.

15. Match the network with the correct IP and prefix to satisfy host requirements: Network A (100 Hosts), Network B (50 Hosts), Network C (25 Hosts), Network D (2 Hosts).

100 hosts needs 7 bits (/25). 50 hosts needs 6 bits (/26). 25 hosts needs 5 bits (/27). 2 hosts (Point-to-Point) needs 2 bits (/30).

16. School A (209.165.201.10) configures an end device to communicate with School B (192.168.25.10) over the internet without a VPN. The administrator knows this will not work. Why?

192.168.x.x is a private IP. Internet routers will drop it instantly because private IPs cannot be routed over the public internet.

17. Which three addresses are valid public addresses?

Option B contains Private IPs (RFC 1918) which are not public. Option A contains valid, routable public IPs.

18. A message is sent to all hosts on a remote network. Which type of message is it?

A directed broadcast targets the broadcast IP of a specific remote network, instead of just flooding your own local switch (limited broadcast).

19. A company has network 192.168.1.64 with mask 255.255.255.192. They want to create two subnetworks for 10 hosts and 18 hosts respectively. Which two networks achieve this?

18 hosts requires a /27 (30 hosts). 10 hosts requires a /28 (14 hosts). So 192.168.1.64/27 and 192.168.1.96/28 perfectly split the /26 block.

20. Which address is a valid IPv6 link-local unicast address?

IPv6 Link-Local addresses ALWAYS begin with FE80::/10 (ranging from FE80 to FEBF).

21. Which is the shortest abbreviation for: 3FFE:1044:0000:0000:00AB:0000:0000:0057?

You can drop leading zeros, and you can only use the double colon (::) ONCE. The largest block of zeros is at the end, so 3FFE:1044:0:0:AB::57 is best.

22. A network admin receives IPv6 prefix 2001:DB8::/48 for subnetting. Assuming the interface ID starts at bit 64, how many subnets can be created?

From /48 to /64 gives you 16 bits for subnetting. 2^16 = 65,536 subnets.

23. Given IPv6 prefix 2001:db8::/48, what will be the LAST subnet created if the subnet prefix is changed to /52?

Moving from /48 to /52 borrows 4 bits (one hex character). The 16 subnets go from 0 to F. The last subnet uses 'F', so it's f000::/52.

24. Consider the following range of addresses: 2001:0DB8:BC15:00A0:: to 00AF::. What is the prefix-length?

The addresses share 2001:0DB8:BC15:00A in common. This is 15 hex characters. 15 * 4 bits = 60 bits.

25. What type of IPv6 address is FE80::1?

Link-local IPv6 addresses start with FE80::/10.

26. A company deploys an IPv6 hierarchy: 2001:DB8:1234:0000::/64. The "site" is 3 hex characters, and the "sub-site" is 1 hex character. What is the maximum number of subnets per sub-site?

One hexadecimal character represents 4 bits. 2^4 = 16 different values (0 through F).

27. What is used in the EUI-64 process to create an IPv6 interface ID?

EUI-64 splits your 48-bit MAC address in half and stuffs "FFFE" in the middle to make it 64-bits.

28. What is the prefix for the host address 2001:DB8:BC15:A:12AB::1/64?

A /64 indicates the first 64 bits (first four hextets) are the network prefix.

29. An IPv6 enabled device sends a data packet with the destination address FF02::1. What is the target of this packet?

FF02::1 is the IPv6 "All Nodes" multicast address. FF02::2 is for "All Routers".

30. Match the IPv6 address with the IPv6 address type.

::1 is loopback. FF02::1 is All-Nodes Multicast. 2001:: is Global Unicast.

31. Which IPv6 prefix is reserved for communication between devices on the same link?

Link-local addresses start with FE80:: and are only routable on the local physical link.

32. Which type of IPv6 address refers to any unicast address that is assigned to multiple hosts?

Anycast assigns one IP to multiple servers. The router delivers the packet to whichever server is closest.

33. What are two types of IPv6 unicast addresses? (Choose two)

Loopback and Link-local are both types of Unicast. IPv6 does not have broadcast addresses.

34. Which service provides dynamic global IPv6 addressing to end devices without using a server that keeps a record?

SLAAC is "Stateless". It allows PCs to invent their own IPs using the router's prefix without relying on a central tracking server.

35. Which protocol supports SLAAC for dynamic assignment of IPv6 addresses?

SLAAC relies on Router Advertisement (RA) messages, which are a function of ICMPv6 Neighbor Discovery.

36. Three methods allow IPv4 and IPv6 to co-exist. Match the definition: "The IPv6 packet is transported inside an IPv4 packet."

Wrapping one packet inside another to cross an incompatible network is called Tunneling.

37. A technician uses the ping 127.0.0.1 command. What is the technician testing?

Pinging the loopback tests if your computer's internal networking software is functioning. It doesn't test the physical cable.

38. An administrator uses the tracert command from PC1 to troubleshoot connectivity to PC2. Where should the administrator begin troubleshooting?

If tracert fails immediately after the first hop, the issue is likely between your PC and the Default Gateway.

39. Which protocol is used by the traceroute command to send and receive echo-requests and echo-replies?

Ping and Traceroute both rely entirely on the ICMP protocol.

40. Which ICMPv6 message is sent when the IPv6 hop limit field of a packet is decremented to zero?

If a packet loops too many times and its TTL / Hop Limit hits 0, the router drops it and sends a "Time Exceeded" message.

41. A user executes a traceroute over IPv6. At what point would a router drop the packet?

Traceroute purposefully sends packets with low Hop Limits to force routers to drop them and reply, revealing their IP addresses.

42. What is the purpose of ICMP messages?

ICMP is the diagnostic protocol of the internet. It provides feedback on errors (like Host Unreachable or Time Exceeded).

43. What source IP address does a router use by default when the traceroute command is issued?

By default, routers source their pings and traceroutes using the IP address of the physical port the packet exits out of.

44. Match the description with the appropriate IP address.

169.254.x.x is APIPA (IPv4 Link Local). 127.0.0.1 is Loopback.

45. A user pings 192.135.250.103 and receives a response code of 1. What does this represent?

In IPv4 ICMP Destination Unreachable messages: Code 0 = Net Unreachable, Code 1 = Host Unreachable.

46. Which subnet would include the address 192.168.1.96 as a usable host address?

The .64/26 subnet ranges from .64 to .127. The address .96 falls perfectly into this range.

47. A host is transmitting a broadcast. Which host or hosts will receive it?

A broadcast (FF:FF:FF:FF:FF:FF) is delivered to every single host on the local subnet.

48. A host is transmitting a unicast. Which host or hosts will receive it?

Unicast means communication directly from ONE device to exactly ONE other device.

49. A user issues a ping 2001:db8:FACE:39::10 and receives a response that includes a code of 3. What does this code represent?

In ICMPv6 Destination Unreachable messages: 0 = No route, 1 = Admin Prohibited, 2 = Beyond Scope, 3 = Address Unreachable, 4 = Port Unreachable.

50. A host is transmitting a multicast. Which host or hosts will receive it?

Multicast is used to send data to a specific, subscribed GROUP of devices.

51. Which is the compressed format of the IPv6 address 2001:0db8:0000:0000:0000:a0b0:0008:0001?

Remove leading zeros, and replace the largest block of zeros with `::`.

52. Which is the compressed format of the IPv6 address fe80:09ea:0000:2200:0000:0000:0fe0:0290?

Replace the longest block of zeros (the two 0000s in the middle) with `::`, and drop leading zeros on the rest.

53. Which is the compressed format of the IPv6 address 2002:0042:0010:c400:0000:0000:0000:0909?

Drop leading zeros (0042 -> 42), and use `::` for the block of all zeros.

54. A user pings 2001:db8... and receives an ICMPv6 response code of 2. What does this represent?

In ICMPv6: 0 = No route, 1 = Admin Prohibited, 2 = Beyond Scope, 3 = Address Unreachable, 4 = Port Unreachable.

55. A user pings 10.10.14.67 (IPv4) and receives a response code of 0. What does this represent?

In IPv4 ICMP Destination Unreachable messages: Code 0 = Network Unreachable.

56. A user pings an IPv6 address and receives an ICMPv6 response code of 4. What does this represent?

In ICMPv6 Destination Unreachable messages: Code 4 = Port Unreachable.
