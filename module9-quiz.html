Module 9 Quiz: FHRP Concepts

1. What is the purpose of HSRP?
Correct Answer: It provides a continuous network connection when a router fails.

Deep Dive: HSRP (Hot Standby Router Protocol) is a First Hop Redundancy Protocol (FHRP). If a company only has one router and it loses power, the entire office loses internet. HSRP fixes this by allowing two physical routers to share a single "Virtual" IP address. If the main router dies, the backup router instantly takes over the virtual IP, ensuring continuous connectivity without the users ever noticing.

  • A. transition to forwarding state: (Incorrect) This describes PortFast.
  • B. prevents malicious hosts: (Incorrect) This describes security features like Port Security or dropping DTP.
  • C. continuous network connection: (Correct) The exact definition of default gateway redundancy.
  • D. prevents rogue switch: (Incorrect) This describes Root Guard or BPDU Guard.
2. Which nonproprietary protocol provides router redundancy for a group of routers which support IPv4 LANs?
Correct Answer: VRRPv2

Deep Dive: There are three main FHRPs you need to memorize. HSRP and GLBP are strictly Cisco-proprietary (they only work on Cisco equipment). If you have a network with a mix of Cisco, Juniper, and HP routers, you MUST use the open-standard (nonproprietary) protocol, which is VRRP (Virtual Router Redundancy Protocol).

  • A & C. GLBP / HSRP: (Incorrect) These are highly effective, but they are Cisco-proprietary.
  • B. SLB: (Incorrect) Server Load Balancing is for distributing traffic to servers, not routing redundancy.
  • D. VRRPv2: (Correct) The industry standard, nonproprietary redundancy protocol for IPv4.
3. A network administrator is analyzing first-hop router redundancy protocols. What is a characteristic of VRRPv3?
Correct Answer: It supports IPv6 and IPv4 addressing.

Deep Dive: VRRP version 2 only supported IPv4. Because the world is transitioning to IPv6, the IEEE released VRRPv3, which natively supports both IPv4 and IPv6 redundancy.

  • A. Works with HSRP: (Incorrect) VRRP and HSRP are competing protocols; they do not work together.
  • B. Allows load balancing: (Incorrect) VRRP is Active/Standby. Only GLBP does true load balancing.
  • C. Supports IPv6 and IPv4: (Correct) The primary upgrade feature of version 3.
  • D. Cisco proprietary: (Incorrect) VRRP is an open IEEE standard.
4. What is a potential disadvantage when implementing HSRP as compared to GLBP?
Correct Answer: HSRP does not provide load balancing with multiple active routers.

Deep Dive: HSRP operates in an "Active / Standby" state. If you buy two $10,000 routers, HSRP forces Router B to sit completely idle doing absolutely nothing until Router A dies. That is a waste of money! GLBP (Gateway Load Balancing Protocol) solves this by operating in "Active / Active", splitting the network traffic 50/50 across both routers simultaneously.

  • A. does not provide load balancing: (Correct) HSRP leaves the backup router sitting idle. GLBP utilizes all routers at the same time.
  • B. does not support IPv6: (Incorrect) Newer versions of HSRP support IPv6.
  • C. multivendor environment: (Incorrect) Neither HSRP nor GLBP support multivendor environments; they are both Cisco proprietary.
5. A network engineer is configuring a LAN with a redundant first hop to make better use of the available network resources. Which protocol should the engineer implement?
Correct Answer: GLBP

Deep Dive: The key phrase in this question is "make better use of available network resources." By implementing GLBP, the engineer ensures that all redundant routers are actively forwarding packets at the same time (Load Balancing), rather than leaving backup routers sitting idle like HSRP and VRRP do.

6. When first hop redundancy protocols are used, which two items will be shared by a set of routers that are presenting the illusion of being a single router? (Combined)
Correct Answer: MAC address AND IP address

Deep Dive: To successfully trick 50 office PCs into thinking there is only one default gateway, the two physical routers must create a "Virtual Router" ghost between them. For the PCs to talk to this ghost, it must be assigned a Virtual IP Address (so the PCs can ping it) and a Virtual MAC Address (so the PCs can send Ethernet frames to it).

  • A. MAC address AND IP address: (Correct) The two necessary addresses shared by the FHRP group.
  • B, C, D. Static route / Hostname / BID: (Incorrect) Hostnames are just local labels. BIDs are used in STP switching, not Layer 3 routing.
7. In FHRP terminology, what represents a set of routers that present the illusion of a single router to hosts?
Correct Answer: Virtual router

Deep Dive: When you configure HSRP, you group physical Router A and physical Router B together. In the configuration, you literally type the command standby 1 ip 192.168.1.254. This creates a "Ghost" or Virtual Router that hovers between them. All the PCs point to the Virtual Router as their gateway.

  • A. Virtual router: (Correct) The illusionary entity created by FHRP.
  • B & C. Forwarding / Standby router: (Incorrect) These describe the *physical* roles the real routers are playing behind the scenes.
  • D. Default gateway: (Incorrect) While the virtual router *acts* as the gateway, "Virtual Router" is the exact FHRP terminology.
8. A user needs to add redundancy to the routers in a company. What are the three options the user can use? (Combined)
Correct Answer: GLBP, VRRP, AND HSRP

Deep Dive: There are only three First Hop Redundancy Protocols (FHRPs) you need to know for the CCNA: HSRP (Cisco Active/Standby), VRRP (Open Standard Active/Standby), and GLBP (Cisco Active/Active Load Balancing).

  • A. GLBP, VRRP, AND HSRP: (Correct) The three valid Layer 3 FHRPs.
  • B, C, D. STP / RAID / IPFIX: (Incorrect) STP is for Layer 2 switch redundancy. RAID is for hard drive storage redundancy. IPFIX is for monitoring traffic flow.
9. Which two protocols provide gateway redundancy at Layer 3? (Combined)
Correct Answer: VRRP AND HSRP

Deep Dive: Redundancy happens at two different OSI layers. If you want redundancy for your physical cabling and switches (Layer 2), you use Spanning Tree Protocol (STP, PVST, RSTP). If you want redundancy for your IP Routing and Default Gateways (Layer 3), you use FHRPs like HSRP and VRRP.

  • A, B, D. Involve STP/PVST: (Incorrect) Spanning Tree Protocols operate at Layer 2 (MAC addresses), not Layer 3 (IP addresses).
  • C. VRRP AND HSRP: (Correct) These are Layer 3 routing redundancy protocols.
10. A network administrator is overseeing the implementation of first hop redundancy protocols. Which two protocols are Cisco proprietary? (Combined)
Correct Answer: GLBP AND HSRP

Deep Dive: Cisco invented HSRP to solve the gateway redundancy problem. Later, they invented GLBP to add load-balancing capabilities. Because Cisco invented both of these, they are proprietary and will only work if all the routers in the group are Cisco brands.

  • B. GLBP AND HSRP: (Correct) The two Cisco-owned protocols.
  • A, C, D. Include VRRP: (Incorrect) VRRP is an open IEEE standard created so non-Cisco routers could also have redundancy.
11. Which statement describes a characteristic of GLBP?
Correct Answer: It provides automatic rerouting if any router in the group fails.

Deep Dive: GLBP shares the traffic load across all routers in the group. If one of those active routers catches on fire and fails, GLBP seamlessly and automatically reroutes that router's portion of the traffic to the surviving routers without the PCs ever dropping a packet.

  • A. automatic rerouting: (Correct) True of all FHRPs, but explicitly true for GLBP's redundant load-balancing nature.
  • B. multiple virtual IP addresses: (Incorrect) GLBP uses a single Virtual IP address, but distributes multiple Virtual MAC addresses to clients.
  • C. no support for IPv6: (Incorrect) GLBP fully supports IPv6.
  • D. maximum of four gateways limit: (Incorrect) GLBP can load balance across up to 4 routers (AVFs), but there is no strict total limit to the number of gateways in the standby group overall. (However, A is the definitively stated answer in Cisco curriculum).
12. A network administrator is analyzing the features that are supported by different first-hop router redundancy protocols. Which statement is a feature that is associated with GLBP?
Correct Answer: GLBP allows load balancing between routers.

Deep Dive: The very name tells you its main feature: Gateway Load Balancing Protocol. While HSRP and VRRP only use one active router at a time, GLBP utilizes all of your expensive routing hardware simultaneously.

  • A. GLBP allows load balancing: (Correct) The primary advantage of GLBP.
  • B. uses a virtual router master: (Incorrect) "Master" is the specific terminology used by VRRP. HSRP uses "Active", and GLBP uses "Active Virtual Gateway (AVG)".
  • C. is nonproprietary: (Incorrect) GLBP is strictly Cisco-proprietary.
  • D. works with VRRP: (Incorrect) They are entirely different, competing protocols.
