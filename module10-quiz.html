1. What two protocols are supported on Cisco devices for AAA communications? (Combined)
Correct Answer: RADIUS AND TACACS+

Deep Dive: In a Server-Based AAA setup, the router does not store usernames and passwords locally. It forwards login requests to a centralized security server. The two standard languages (protocols) the router uses to talk to this server are RADIUS (the open standard) and TACACS+ (the Cisco proprietary, fully encrypted standard).

  • A. HSRP & LLDP: (Incorrect) HSRP is for router redundancy. LLDP is for neighbor discovery.
  • B. RADIUS AND TACACS+: (Correct) The two primary AAA authentication protocols.
  • C. VTP & CDP: (Incorrect) VTP syncs VLANs. CDP is Cisco's neighbor discovery.
  • D. SSH & HTTPS: (Incorrect) These are used to securely access the router, but they do not query backend AAA servers.
2. Which service is enabled on a Cisco router by default that can reveal significant information about the router and potentially make it more vulnerable to attack?
Correct Answer: CDP

Deep Dive: CDP (Cisco Discovery Protocol) is a helpful tool that makes Cisco devices broadcast information about themselves (like OS version, device model, and IP addresses) so network admins can draw topology maps. However, because it is enabled by default globally, if it isn't disabled on user-facing ports, a hacker can run a packet sniffer and immediately learn exactly what Cisco vulnerabilities to exploit.

  • A. LLDP: (Incorrect) While LLDP does the exact same thing, it is an open standard and is typically NOT enabled by default on Cisco routers.
  • B & C. HTTP / FTP: (Incorrect) Application layer protocols not used for neighbor discovery.
  • D. CDP: (Correct) The Cisco proprietary protocol enabled by default that leaks reconnaissance data to attackers.
3. When security is a concern, which OSI Layer is considered to be the weakest link in a network system?
Correct Answer: Layer 2

Deep Dive: Security is only as strong as its weakest link. For decades, admins focused on securing Layer 3 (Routers/Firewalls) and Layer 7 (Applications). They completely ignored Layer 2 (The Data Link Layer / Switching). Because switches inherently trust physical connections, Layer 2 is highly vulnerable to inside attacks like MAC Flooding, ARP Spoofing, and VLAN Hopping.

  • A, B, C. Layers 4, 3, 7: (Incorrect) These layers usually have extensive security mechanisms (like VPNs, Access Control Lists, and HTTPS).
  • D. Layer 2: (Correct) The switching layer is historically the most vulnerable "weakest link" on local area networks.
4. Which Layer 2 attack will result in a switch flooding incoming frames to all ports?
Correct Answer: MAC address overflow

Deep Dive: Switches use a MAC Address Table (CAM table) to remember which PC is on which port. In a MAC address overflow (or MAC Flooding) attack, a hacker uses a tool like *macof* to blast 100,000 fake MAC addresses at the switch in seconds. The table fills up, and the switch panics. Instead of dropping traffic, it "fails open" and starts behaving like a dumb Hub—flooding every single frame out of all ports, allowing the hacker to sniff everyone's traffic.

  • A. STP manipulation: (Incorrect) Used to alter the root bridge and intercept traffic, but doesn't cause raw flooding.
  • B. ARP poisoning: (Incorrect) Tricks PCs into sending traffic to the hacker (Man-in-the-Middle).
  • C. IP address spoofing: (Incorrect) Faking an IP to bypass ACLs.
  • D. MAC address overflow: (Correct) Fills the CAM table, forcing the switch to flood unknown unicast frames.
5. Why is authentication with AAA preferred over a local database method?
Correct Answer: It provides a fallback authentication method if the administrator forgets the username or password.

Deep Dive: If you only configure local passwords on a router, and you forget them, you are completely locked out and have to perform a factory password recovery. With AAA, you can set up a "Method List". This allows you to say: "Try to log me in using the RADIUS server first. If the server is offline or I forgot my RADIUS password, fallback to the local backup database."

  • A. uses less bandwidth: (Incorrect) AAA actually uses *more* bandwidth because it has to send packets to a server.
  • B. provides a fallback authentication method: (Correct) The modular nature of AAA lets you chain multiple login methods together for safety.
  • C. different password for each line: (Incorrect) You can do this with local databases too.
  • D. requires login combination: (Incorrect) You can enforce this without AAA.
6. In a server-based AAA implementation, which protocol will allow the router to successfully communicate with the AAA server?
Correct Answer: RADIUS

Deep Dive: When a router needs to ask a central server if a user's password is correct, it uses either RADIUS or TACACS+. (Note: *TACACS* is an ancient, obsolete protocol. *TACACS+* is the modern Cisco version. Since TACACS+ is not an option here, RADIUS is the only correct answer.)

  • A. SSH: (Incorrect) Secure Shell is used by the *human* to talk to the *router*, not the router to the server.
  • B. RADIUS: (Correct) The industry standard open-source AAA protocol.
  • C. 802.1x: (Incorrect) This is a port-authentication standard that *relies* on RADIUS, but is not the server communication protocol itself.
  • D. TACACS: (Incorrect) A legacy protocol replaced entirely by TACACS+.
7. Which Cisco solution helps prevent MAC and IP address spoofing attacks?
Correct Answer: IP Source Guard

Deep Dive: If a hacker manually changes their computer's IP or MAC address to bypass security filters, it's called spoofing. IP Source Guard (IPSG) stops this by checking every packet against the DHCP Snooping binding table. If the source IP/MAC doesn't match what the DHCP server originally handed out to that specific port, IPSG drops the traffic instantly.

  • A. IP Source Guard: (Correct) Explicitly prevents IP and MAC address spoofing.
  • B. Dynamic ARP Inspection: (Incorrect) DAI prevents ARP poisoning (Man-in-the-Middle) attacks, not general IP spoofing.
  • C. DHCP Snooping: (Incorrect) Prevents rogue DHCP servers.
  • D. Port Security: (Incorrect) Prevents MAC flooding attacks, but does not check Layer 3 IP addresses.
8. What is the purpose of AAA accounting?
Correct Answer: To collect and report application usage

Deep Dive: The AAA framework has three distinct jobs. Authentication checks identities. Authorization checks permissions. Accounting is the logging phase. It tracks how long a user was logged in, how much bandwidth they used, and what commands they typed. This data is used for billing and security auditing.

  • A. prove users are who they say they are: (Incorrect) This is Authentication.
  • B & C. determine operations/resources: (Incorrect) This is Authorization.
  • D. collect and report usage: (Correct) The definition of Accounting.
9. Which Layer 2 attack will result in legitimate users not getting valid IP addresses?
Correct Answer: DHCP starvation

Deep Dive: A DHCP server has a limited "pool" of IP addresses it can hand out. In a DHCP Starvation attack, a hacker uses a tool to generate thousands of fake MAC addresses, requesting a new IP for every single one. Within seconds, the DHCP pool is completely exhausted, and real employees can't get an IP address to join the network.

  • A. IP address spoofing: (Incorrect) Faking an identity.
  • B. DHCP starvation: (Correct) Exhausts the IP pool, creating a Denial of Service.
  • C. ARP spoofing: (Incorrect) Redirects traffic to the attacker.
  • D. MAC address flooding: (Incorrect) Fills the switch's CAM table, causing frame flooding.
10. Which three Cisco products focus on endpoint security solutions? (Combined)
Correct Answer: Email Security Appliance, Web Security Appliance, AND NAC Appliance

Deep Dive: Security is split into two categories: "Network" security (protecting the wires and routers) and "Endpoint" security (protecting the user's laptop/email). The Cisco ESA (Email), WSA (Web), and NAC (Network Admission Control) are specifically designed to filter malware, block bad websites, and check laptops for antivirus software before letting them on the network.

  • A. Email, Web, and NAC: (Correct) These focus directly on securing end-user devices (endpoints).
  • B, C, D. Include VPN/ASA/IPS: (Incorrect) ASA Firewalls, VPNs, and IPS sensors are considered "Network Infrastructure" security devices, not endpoint devices.
11. True or False? In the 802.1X standard, the client attempting to access the network is referred to as the supplicant.
Correct Answer: True

Deep Dive: The 802.1X protocol uses three specific roles. The user's PC or laptop that is begging for access to the switch port is called the Supplicant. The Switch is the Authenticator. The RADIUS server is the Authentication Server.

12. What is involved in an IP address spoofing attack?
Correct Answer: A legitimate network IP address is hijacked by a rogue node.

Deep Dive: Spoofing is the act of falsifying your identity. In an IP Spoofing attack, a hacker manually changes their computer's IP address to match the IP address of a trusted machine (like the CEO's computer or a Server). This allows the hacker to bypass IP-based Access Control Lists (ACLs) and hijack traffic.

  • A. Rogue DHCP server: (Incorrect) This is a DHCP Spoofing attack.
  • B. Bogus DHCPDISCOVER messages: (Incorrect) This is a DHCP Starvation attack.
  • C. Legitimate IP address is hijacked: (Correct) The exact definition of IP Address Spoofing.
  • D. Rogue node replies to ARP request: (Incorrect) This is ARP Poisoning (ARP Spoofing).
13. What three services are provided by the AAA framework? (Combined)
Correct Answer: Accounting, Authorization, AND Authentication

Deep Dive: The acronym AAA strictly stands for:
1. Authentication (Proving who you are).
2. Authorization (Checking what you are allowed to do).
3. Accounting (Logging everything you did).

14. Because of implemented security controls, a user can only access a server with FTP. Which AAA component accomplishes this?
Correct Answer: Authorization

Deep Dive: Once a user types their password correctly (Authentication), the AAA server looks up their profile to see their privileges. If the profile says "This user is only permitted to use FTP, block HTTP and SSH," that restrictive rule application is the definition of Authorization.

  • A. Authorization: (Correct) Determines which network resources and operations are permitted.
  • B. Accounting: (Incorrect) Logs the fact that the user used FTP.
  • C. Authentication: (Incorrect) Verifies the user's password.
15. What mitigation plan is best for thwarting a DoS attack that is creating a MAC address table overflow?
Correct Answer: Enable port security.

Deep Dive: A MAC address table overflow attack (MAC Flooding) works by sending 100,000 fake MAC addresses into a single switch port. The easiest way to stop this is to configure Port Security. You can tell the switch, "Only allow a maximum of 2 MAC addresses on this port." The moment the hacker sends the 3rd fake MAC address, the switch will instantly shut the port down, thwarting the attack.

  • A & B. Unused VLANs / Disable DTP: (Incorrect) These steps mitigate VLAN Hopping attacks, not MAC flooding.
  • C. Disable STP: (Incorrect) You should NEVER disable STP, as it prevents physical loops.
  • D. Enable port security: (Correct) The definitive countermeasure for MAC flooding.
