Practice Exam: Communicating Between Networks

Select your answers below. Tap any blue dashed word to understand it. When finished, click "Check Results" at the very bottom!

1. Which information is used by routers to forward a data packet toward its destination?

Routers operate at Layer 3 and strictly use the destination IP address to determine the path from their routing table.

2. A computer has to send a packet to a destination host in the same LAN. How will the packet be sent?

If two computers are on the same LAN (subnet), they don't need a router. The switch handles it directly using MAC addresses.

3. A router receives a packet from Gigabit 0/0 and determines it needs to be forwarded out Gigabit 0/1. What will the router do next?

It already looked at the routing table. To physically send the packet out of G0/1, it MUST encapsulate it into a new Layer 2 frame.

4. Which IPv4 address can a host use to ping the loopback interface?

127.0.0.1 is the universal IPv4 loopback address. Pinging it tests your PC's internal TCP/IP stack.

5. A computer can access devices on the same network but cannot access devices on other networks. What is the probable cause?

To leave your local network, you need a working default gateway (router). If that's wrong, you are trapped locally.

6. Which statement describes a feature of the IP protocol?

IP is an "unreliable" protocol. It relies on Layer 4 (like TCP) to check for missing packets and re-order them.

7. Why is NAT not needed in IPv6?

NAT was invented because we ran out of IPv4 addresses. IPv6 provides so many addresses that every device can have a public one.

8. Which parameter does the router use to choose the path to the destination when multiple routes are available?

A metric represents the "cost" of taking a path. A router always chooses the lowest cost (lowest metric).

9. What are two services provided by the OSI network layer?

The Network Layer (Layer 3) encapsulates data into packets and routes them. Error/collision detection is Layer 2.

10. Within a production network, what is the purpose of configuring a switch with a default gateway address?

A switch only needs a gateway so *it* can reply to a remote administrator trying to SSH into it from a different network.

11. What is a basic characteristic of the IP protocol?

IP is connectionless, meaning it sends data without verifying if the receiver is ready or listening.

12. Which field in the IPv4 header is used to prevent a packet from traversing a network endlessly?

Every router decreases the TTL by 1. If it hits zero, the packet is killed to prevent infinite routing loops.

13. What is one advantage that the IPv6 simplified header offers over IPv4?

IPv6 removed several fields (like checksums) to make the header streamlined, resulting in highly efficient packet handling.

14. What IPv4 header field identifies the upper layer protocol carried in the packet?

The Protocol field identifies if the payload inside is TCP, UDP, ICMP, etc.

15. In routing, if a packet's destination network has NO exact entry in the routing table, what will the router do?

If a specific network isn't listed, the router forwards it to the Default Route (Gateway of Last Resort). E.g., your home router sending traffic to the ISP.

16. What information does the loopback test provide?

Pinging 127.0.0.1 tests the internal software and network card. It does NOT test the physical cable or internet connection.

17. What routing table entry has a next hop address associated with a destination network?

Remote routes require jumping through another router to be reached, hence they have a "next hop" IP address. Local routes do not.

18. How do hosts ensure that their packets are directed to the correct network destination?

Your PC actually maintains its own tiny local routing table containing a route to the loopback, local network, and local default route!

19. When transporting data from real-time applications (streaming audio/video), which IPv6 field informs routers to maintain the same path?

The 20-bit Flow Label tags real-time streams so routers prioritize keeping them on the exact same path to prevent lag/re-ordering.

20. What statement describes the function of the Address Resolution Protocol (ARP)?

When a PC wants to send data, it knows the IP, but needs the MAC. ARP asks "Who has this IP?" on the local network.

21. Under which two circumstances will a switch flood a frame out of every port except the incoming port?

If a switch doesn't know where a MAC address is, or if the destination is FF:FF:FF:FF:FF:FF (broadcast), it panics and floods it everywhere.

22. Which statement describes the treatment of ARP requests on the local link?

ARP requests are sent as a broadcast. This means ALL devices on the local link must receive and process the request.

23. Which destination address is used in an ARP request frame?

This is the Layer 2 Broadcast MAC address. It ensures every single device on the switch receives the ARP request.

24. A network technician issues the arp -d * command on a PC. What is the result?

The -d stands for delete. This clears the cache to force the PC to re-learn updated MAC addresses.

25. PC1 sends a frame addressed to PC3. However, PC3's MAC address is NOT in the switch's MAC address table. What will the switch do?

Because the switch doesn't know where PC3 is, it floods the frame out of every port except the one it received it from to ensure PC3 gets it.

26. Which two types of IPv6 messages are used in place of ARP for address resolution?

IPv6 does NOT use ARP broadcasts! Instead, it uses ICMPv6 Neighbor Discovery via Solicitation and Advertisement messages.

27. What is the aim of an ARP spoofing attack?

In ARP Spoofing, a malicious host lies to other PCs so they map the Router's IP address to the Hacker's MAC address.

28. PC1 connects to a remote File Server and sends an ARP request. Which MAC address will PC1 receive in the ARP reply?

Because the Server is on a remote network, PC1 must send the frame to its Default Gateway. Therefore, PC1 ARPs for the router's MAC address.

29. Where are IPv4 address to Layer 2 Ethernet address mappings maintained on a host computer?

The ARP cache stores the mapping of IPs to MAC addresses directly on your PC.

30. What important information is examined in the Ethernet frame header by a Layer 2 device in order to forward the data onward?

A Layer 2 device (switch) strictly uses the DESTINATION MAC address to determine which port to send the data out of.

31. Which command is used to configure a message that displays after accessing the router?

The Message of the Day (MOTD) banner displays a warning/greeting when users connect.

32. What is the fastest way a network administrator could test whether the banner is properly configured?

Typing 'exit' until you leave the router entirely and then pressing Enter will instantly show the login banner without a slow reboot.

33. Which remote access method relies on an active network connection and uses encryption?

SSH uses strong password authentication and encryption. Telnet is unencrypted. Console is local/physical, not remote network access.

34. What is the correct sequence of the three major phases of a Cisco router bootup process?

Hardware test (POST) -> Load OS (from Flash) -> Load saved settings (from NVRAM).

35. In which device mode is the service password-encryption command entered?

It is entered in Global Configuration Mode (R1(config)#).

36. Which two functions are properties of NVRAM?

NVRAM is non-volatile. The startup configuration file is preserved here even if the router loses power.

37. A router boots and enters setup mode. What is the reason for this?

If a router cannot find a saved `startup-config` in NVRAM, it assumes it is a brand-new router and enters interactive "setup mode".

38. The global configuration command ip default-gateway 172.16.100.1 is applied to a switch. What is the effect?

A switch needs a default gateway purely so it knows how to reply to an admin trying to SSH into it from a remote network.

39. What happens when the transport input ssh command is entered on the switch vty lines?

Applying this to the VTY lines forces the switch to accept only secure, encrypted SSH connections, blocking unencrypted Telnet.

40. A user PC has successfully transmitted packets to www.cisco.com. Which IP address does the user PC target in order to forward its data off the local network?

Any time a PC needs to leave the local network, it forwards the packet to the MAC address of its Default Gateway (Router).

41. Which prompt represents Privileged EXEC mode where you can save the configuration?

The '#' prompt indicates Privileged EXEC mode, where you use `copy running-config startup-config`.

42. Which three commands are used to set up secure access to a router through a connection to the console interface?

To secure the physical console port, you enter line config mode, set the password, and command it to prompt for `login`.

43. What is a description of the default gateway address?

The default gateway is the FIRST Layer 3 device (router interface) that connects directly to your local network.

44. Which two functions are primary functions of a router?

A router accesses its routing table to determine the best path (path selection) and then pushes the packet out (packet forwarding).

45. What is the effect of using the copy running-config startup-config command on a router?

This command saves the current RAM configuration into the permanent NVRAM so it survives a reboot.

46. What will happen if the default gateway address is incorrectly configured on a host?

Local communication works fine without a gateway. But to leave the LAN, the PC needs the correct router IP.

47. What are two potential network problems that can result from ARP operation?

ARP relies on broadcasts (which cause delays) and has no security (allowing ARP Spoofing).

48. Packet Tracer Verification: Using `show ip interface brief`, which interfaces are active and operational?

Active interfaces are represented by the value "up" in both the "Status" and "Protocol" columns.

49. Which term describes a field in the IPv4 packet header used to identify the next level protocol?

The protocol field identifies if the payload is TCP, UDP, ICMP, etc.

50. Which term describes a field in the IPv4 packet header that contains an 8-bit binary value used to determine the priority of each packet?

DS (or DSCP) is used for Quality of Service (QoS) to prioritize voice/video traffic over normal data.

51. Which term describes a field in the IPv4 packet header that contains a 32-bit binary value associated with an interface on the sending device?

The sender's IP address.

52. Which term describes a field in the IPv4 packet header used to detect corruption in the IPv4 header?

Used to determine if any errors were introduced during transmission. (Removed in IPv6!)

53. A network administrator is connecting a new host to the Payroll LAN. What IP address would be configured as the default gateway on the new host? 
RTR1(config)# interface gi0/1
RTR1(config-if)# description Connects to the Marketing LAN
RTR1(config-if)# ip address 10.27.15.17 255.255.255.0
RTR1(config)# interface gi0/0
RTR1(config-if)# description Connects to the Payroll LAN
RTR1(config-if)# ip address 10.27.14.148 255.255.255.0

The default gateway is the router interface directly connected to its LAN (Payroll LAN = gi0/0 = 10.27.14.148).

54. Which term describes a field in the IPv4 packet header that contains a unicast, multicast, or broadcast address?

The destination IP can be a single PC (unicast), a group (multicast), or everyone (broadcast).

55. Which term describes a field in the IPv4 packet header used to limit the lifetime of a packet?

Prevents endless routing loops by dropping the packet when TTL hits 0.

56. Which term describes a field in the IPv4 packet header that contains a 4-bit binary value set to 0100?

0100 in binary equals the number 4 (IPv4). For IPv6, the version field is set to 0110 (6).

57. What property of ARP causes cached IP-to-MAC mappings to remain in memory longer?

Cached entries naturally stick around until their timeout limit is reached.

58. What property of ARP allows MAC addresses of frequently used servers to be fixed in the ARP table?

Static entries are permanent and immune to ARP Spoofing!

59. What property of ARP allows hosts on a LAN to send traffic to remote networks?

To reach the internet, the PC uses ARP to find the router's MAC address.

61. What property of ARP forces all Ethernet NICs to process an ARP request?

FF-FF-FF-FF-FF-FF is the Layer 2 Broadcast address. It forces every NIC to accept the frame.

62. What property of ARP causes a reply only to the source sending an ARP request?

When the target replies, it uses the original sender's Source MAC as its new Destination MAC to send a Unicast reply directly back.

63. What property of ARP causes the request to be flooded out all ports of a switch except for the port receiving the ARP request?

Because the destination is a broadcast, the switch must flood it everywhere to ensure the correct device hears it.

64. What property of ARP causes the NICs receiving an ARP request to pass the data portion of the Ethernet frame to the ARP process?

The "Type" field tells the computer what protocol is inside the frame. 0x0806 is the hex code specifically for ARP.
