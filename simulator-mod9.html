Lab Objectives

  • Change the HSRP protocol version to version 2.

📜 Cisco CLI Cheat Sheet: Mod 9 HSRP

1. Configure the Active Router (R1)

interface g0/1
standby version 2
Enables HSRP version 2 (required for IPv6 and modern networks).
standby 1 ip 192.168.1.254
Creates HSRP Group 1 and assigns the shared Virtual IP address that PCs will use as their Default Gateway.
standby 1 priority 150
The default priority is 100. Changing it to 150 forces this router to win the election and become Active.
standby 1 preempt
Forces this router to instantly take back the Active role if it reboots and comes back online.

2. Configure the Standby Router (R3)

interface g0/0
standby version 2
standby 1 ip 192.168.1.254
The standby router only needs the version, group, and virtual IP. It uses the default priority of 100, so it automatically becomes the backup.

3. Verification Commands

show standby
Displays detailed HSRP info (Active/Standby IPs, Virtual MAC address, timers).
show standby brief
Shows a quick summary table of HSRP interfaces, groups, priorities, and roles.
