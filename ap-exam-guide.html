Packet Tracer Exam Walkthrough: WLC & APs

If your exam requires configuring an Access Point, you will likely be asked to set up a WLAN using a WLC and secure it with WPA2-Enterprise using a RADIUS server. Follow these exact Packet Tracer clicks.

1
Log into the WLC

You cannot click the AP directly. You must use the Admin PC to access the WLC's web interface.

1. Click the Admin PC.
2. Go to the Desktop tab.
3. Open the Web Browser.
4. Type the WLC's IP Address (e.g., https://192.168.200.254).
🚨 PACKET TRACER TRAP: You MUST type https:// before the IP address. If you just type the IP or use http://, Packet Tracer will give you a "Request Timeout" error and you will be stuck!

2
Create the Dynamic Interface

Before making a Wi-Fi network, the WLC needs a virtual interface assigned to a VLAN so traffic can reach the switch.

1. Click the CONTROLLER tab at the top.
2. Click Interfaces on the left menu.
3. Click the New... button.
4. Name it (e.g., "WLAN-5-Int") and assign the VLAN ID (e.g., "5").
5. Enter the IP Address, Subnet Mask, and Default Gateway given in your exam table.
6. Enter the DHCP Server IP address. Click Apply.

3
Configure the RADIUS Server

Enterprise networks don't use shared passwords (PSK). You must tell the WLC where the Authentication Server is located.

1. Click the SECURITY tab at the top.
2. Click RADIUS > Authentication on the left.
3. Click the New... button.
4. Enter the Server IP Address.
5. Enter the Shared Secret (e.g., "RadiusPW123").
6. Click Apply.

4
Create and Secure the WLAN

Now you create the actual SSID that the laptops will see broadcasting from the AP.

1. Click the WLANs tab at the top.
2. Select "Create New" from the dropdown and click Go.
3. Type the Profile Name and SSID name (Match the exam EXACTLY!).
4. On the General tab, check the Status: [x] Enabled box.
5. On the General tab, change the Interface to the one you made in Step 2.
6. On the Security > Layer 2 tab, choose WPA+WPA2, enable WPA2 Policy, and choose 802.1X.
7. On the Security > AAA Servers tab, select your RADIUS server from the dropdown.
8. Click Apply.
🚨 PACKET TRACER TRAP: If you forget to check the Status: [x] Enabled box on the General tab, the Access Point will never broadcast the signal, and you will fail the lab!

5
Connect the Wireless Host

The AP is now broadcasting the signal. You must connect the laptop using the supplicant software.

1. Click the Laptop device.
2. Go to the Desktop tab.
3. Open the PC Wireless app.
4. Click the Profiles tab (Do NOT use the "Connect" tab for Enterprise!).
5. Click New and name the profile.
6. Select the SSID from the list, click Advanced Setup.
7. Choose Infrastructure Mode.
8. Choose WPA2-Enterprise.
9. Enter the Username and Password provided in your exam instructions.
10. Click Connect to Network.
🚨 PACKET TRACER TRAP: The default "Connect" tab in the PC Wireless app does not support WPA2-Enterprise (it only supports Passphrases). You MUST use the Profiles tab to enter a Username and Password.
