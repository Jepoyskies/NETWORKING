1. Which authentication method stores usernames and passwords in the router and is ideal for small networks?
Correct Answer: local AAA

Deep Dive: AAA (Authentication, Authorization, Accounting) controls who gets into your network. "Local" means you literally type the usernames/passwords into the running config of the router itself. It's fast, but terrible for large networks because you'd have to update passwords on every single router manually.

  • A. local AAA: (Correct) Stores credentials directly in the local device's NVRAM database. Ideal for small networks.
  • B. local AAA over RADIUS: (Incorrect) A trick answer. "Local" means no server. RADIUS is a server protocol. You can't mix them.
  • C/D. server-based AAA: (Incorrect) This relies on an external server (like Cisco ISE). Meant for medium-to-large networks, not small ones.
2. What are two protocols that are used by AAA to authenticate users against a central database of usernames and passwords? (Choose two.)
Correct Answer: TACACS+ and RADIUS

Deep Dive: When you configure Server-Based AAA, your switch needs a "language" to ask the security server "Is this password correct?". Cisco networks use two main protocols for this: RADIUS (open-standard) and TACACS+ (Cisco proprietary and fully encrypted).

  • A. NTP & HTTPS: (Incorrect) NTP synchronizes network time. HTTPS secures web traffic. Neither are AAA server protocols.
  • B. TACACS+ and RADIUS: (Correct) The two industry standards for communicating with a centralized AAA server.
  • C & D. SSH, CHAP, & TACACS+ combined: (Incorrect) SSH secures terminal access. CHAP securely sends a password over a WAN link. TACACS+ combined with SSH is mixing a server protocol with a terminal protocol.
3. What is the result of a DHCP starvation attack?
Correct Answer: Legitimate clients are unable to lease IP addresses.

Deep Dive: A DHCP Starvation attack is like one greedy person grabbing all the free samples so nobody else gets any. The attacker uses a tool (like Gobbler) to rapidly spoof thousands of fake MAC addresses, requesting an IP for each one until the DHCP server's pool is 100% empty (starved).

  • A. Legitimate clients are unable... (Correct) Because the pool is empty, real users get nothing, causing a Denial of Service (DoS).
  • B & C. Rogue DHCP / Incorrect DNS: (Incorrect) These describe DHCP Spoofing, a completely different attack where the hacker sets up their own fake server to intercept traffic.
  • D. IPs are hijacked: (Incorrect) Starvation prevents *new* leases; it rarely steals an IP already actively assigned.
4. What represents a best practice concerning discovery protocols such as CDP and LLDP on network devices?
Correct Answer: Disable both protocols on all interfaces where they are not required.

Deep Dive: CDP (Cisco) and LLDP (Open) are tools that allow devices to "shout" their identity ("Hi, I'm a Cisco Switch running iOS 15.2!"). These broadcasts are unencrypted. If left enabled on ports facing user PCs (Edge ports), a hacker running Wireshark instantly knows your exact equipment model and vulnerabilities.

  • A. Use LLDP rather than CDP: (Incorrect) LLDP leaks the exact same dangerous information as CDP.
  • B. Disable on interfaces where not required: (Correct) Principle of Least Privilege. User PCs do not need to know switch details. Disable it on all edge ports.
  • C. Use default settings: (Incorrect) Cisco defaults enable CDP globally. This is a massive security risk.
  • D. Enable on edge devices: (Incorrect) Edge devices are exactly where you want it disabled to stop hackers from gathering recon data.
5. Which protocol should be used to mitigate the vulnerability of using Telnet to remotely manage network devices?
Correct Answer: SSH

Deep Dive: Admins log into switches remotely via the VTY Lines. Telnet sends everything in pure plaintext. If a hacker intercepts the traffic, they can read your admin password like a book. To fix this, we use Secure Shell (SSH), which establishes a highly encrypted tunnel before you even type your username.

  • A. SCP: (Incorrect) Secure Copy Protocol is for securely transferring *files*, not for remote command-line terminal management.
  • B. SSH: (Correct) SSH is the industry standard encrypted replacement for Telnet.
  • C. TFTP: (Incorrect) Used for sending files. It is unencrypted and has zero authentication.
  • D. SNMP: (Incorrect) Used for monitoring network device health, not for terminal access.
6. Which statement describes the behavior of a switch when the MAC address table is full?
Correct Answer: It treats frames as unknown unicast and floods all incoming frames to all ports within the local VLAN.

Deep Dive: This is the exact mechanism exploited by a MAC Flooding attack. When the switch's memory is full, it can't remember where specific PCs are. It panics and acts like a dumb Hub, broadcasting the frame everywhere so it doesn't drop the data. Crucially, a switch respects VLAN boundaries, so it only floods *within* the specific VLAN the frame belongs to.

  • A. floods to all ports on the switch: (Incorrect) It does not flood to ports belonging to *other* VLANs. VLANs isolate broadcast traffic.
  • B. floods to all ports within the local VLAN: (Correct) This is the precise behavior of a full MAC table.
7. Which feature on a switch makes it vulnerable to VLAN hopping attacks?
Correct Answer: the automatic trunking port feature enabled for all ports by default

Deep Dive: VLAN Hopping allows a hacker in VLAN 10 to send traffic directly to a server in VLAN 20, bypassing the router firewall. The hacker does this by sending fake DTP messages, tricking the switch into turning the hacker's port into a Trunk Link.

  • B. limited memory space: (Incorrect) This causes MAC Flooding, not VLAN Hopping.
  • C. automatic trunking feature: (Correct) DTP is enabled by default on Cisco switches. A hacker acts like a switch, forms a trunk, and gains access to all VLANs. Always use `switchport mode access` to disable it!
8. Which feature or configuration on a switch makes it vulnerable to VLAN double-tagging attacks?
Correct Answer: the native VLAN of the trunking port being the same as a user VLAN

Deep Dive: Double-tagging is a sneaky, unidirectional attack. The hacker hides a second, fake VLAN tag inside their frame. When the first switch receives it, it strips off the first tag (because it's the Native VLAN, which is untagged). The frame is forwarded over the trunk, and the second switch reads the *hidden* tag, forwarding it to the victim VLAN.

  • C. automatic trunking: (Incorrect) This causes basic VLAN Hopping, not Double-Tagging. Double-tagging works *even if trunking is disabled on the user port*.
  • D. native VLAN same as user VLAN: (Correct) The attack only works if the hacker is physically connected to a port that shares the exact same VLAN ID as the trunk's Native VLAN. This is why you must change the Native VLAN to an unused "black hole" number (like 999).
9. Which component of AAA is used to determine which resources a user can access and which operations the user is allowed to perform?
Correct Answer: authorization

Deep Dive: AAA stands for Authentication, Authorization, and Accounting. Think of it like entering a highly secure building.

  • Authentication: Proving *who* you are (Scanning your ID badge).
  • Authorization: (Correct) Determining *what* you can do. The system checks if your ID badge allows you to open the Server Room door, or if you are only allowed in the Lobby.
  • Accounting / Auditing: Keeping a strict logbook of what time you entered the Server Room and what you touched.
10. Which component of AAA allows an administrator to track individuals who access network resources and any changes that are made to those resources?
Correct Answer: accounting

Deep Dive: As explained above, the third "A" in AAA stands for Accounting. After a user logs in (Authentication) and is granted permissions (Authorization), Accounting keeps a detailed, timestamped log of exactly what commands they typed. This is crucial for tracing back security breaches to a specific employee's account.

11. What device is considered a supplicant during the 802.1X authentication process?
Correct Answer: the client that is requesting authentication

Deep Dive: The 802.1X protocol is used for port-based access control (locking down a physical wall jack). It has three specific roles. Think of it like trying to get into an exclusive nightclub: you are the Supplicant (the person asking to get in), the Bouncer at the door is the Authenticator (the Switch), and the guy in the back room checking the VIP list is the Authentication Server (the RADIUS server).

  • A. the client that is requesting authentication: (Correct) The end-user's PC or laptop running the 802.1X client software is known as the Supplicant.
  • B. the switch that is controlling access: (Incorrect) The switch is the Authenticator. It doesn't know your password; it just holds the door shut until the server says you are good.
  • C. the authentication server: (Incorrect) This is the RADIUS/AAA server, not the supplicant.
  • D. the router: (Incorrect) Routers serving as default gateways are not directly involved in the Layer 2 802.1X port-authentication triad.
12. The Fa0/2 interface on switch S1 has been configured with the switchport port-security mac-address 0023.189d.6456 command and a workstation has been connected. What could be the reason that the Fa0/2 interface is shutdown?
Correct Answer: The MAC address of PC1 that connects to the Fa0/2 interface is not the configured MAC address.

Deep Dive: You manually hardcoded a static MAC address (0023.189d.6456) onto this port using Port Security. Because you didn't specify a "maximum" number of MACs, it defaults to 1. If the port shuts down, it means a security violation occurred. The only way that happens here is if a PC with a different MAC address was plugged in.

  • A. crossover cable: (Incorrect) Modern switches have Auto-MDIX and will automatically fix the cable type. It wouldn't trigger a security shutdown.
  • B. Fa0/24 configured with same MAC: (Incorrect) While bad practice, this wouldn't instantly shut down Fa0/2 unless that MAC moved.
  • C. aging command: (Incorrect) Aging just deletes old MAC addresses after a certain time; it doesn't cause violations.
  • D. MAC address of PC1 is not the configured MAC: (Correct) Plugging in an unauthorized device triggers the default violation mode: Shutdown.
13. An IP phone and a PC are connected to port Fa0/2. The admin has the following goals: 1) Allow ONLY these 2 devices. 2) Automatically learn their MAC addresses and save them to the config. 3) Shut down the port if a 3rd device connects. Which commands are required?
Correct Answer: Option C (Enable, Max 2, Sticky)

Deep Dive: To use Port Security, you MUST turn it on first using the base command. Then, since you have two devices (a PC daisy-chained through a VoIP phone), you must change the maximum from the default of 1 to 2. Finally, to have the switch save them to the running config automatically, you use "sticky". Because you want it to shut down on a violation, you don't need a violation command—shutdown is already the default!

  • A. Missing maximum: (Incorrect) Defaults to 1 MAC. Since there are 2 devices, it would shut down immediately.
  • B. Missing base command: (Incorrect) You cannot configure "sticky" or "max 2" if you haven't turned the feature on with the `switchport port-security` base command!
  • C. Base, Max 2, Sticky: (Correct) Turns it on, allows 2 devices, and remembers them. If a 3rd connects, it defaults to shutting down.
  • D. Includes violation restrict: (Incorrect) Restrict drops traffic but keeps the port UP. The prompt specifically asked to "shut down" the port if a 3rd device connects.
14. Port security has been configured on interface Fa0/12. It has a manually configured static MAC. What action will occur when an unauthorized PC1 attaches to the switch port?
Correct Answer: Frames from PC1 will cause the interface to shut down immediately, and a log entry will be made.

Deep Dive: When you enable Port Security, you don't actually *have* to type a violation command. If you skip it, the switch automatically uses the default Violation Mode, which is Shutdown.

  • A. Dropped, no log: (Incorrect) This describes the "Protect" violation mode.
  • B. Dropped, log created, stays up: (Incorrect) This describes the "Restrict" violation mode.
  • C. Interface shuts down immediately with log: (Correct) This is the default "Shutdown" violation mode behavior. The port goes into err-disabled state.
  • D. Forwarded because command is missing: (Incorrect) The switch applies default settings automatically.
15. A company security policy specifies that when a port security violation occurs, packets with unknown source addresses should be dropped and NO notification (syslog) should be sent. Which violation mode should be configured?
Correct Answer: protect

Deep Dive: There are three Violation Modes you must memorize for the CCNA: Shutdown, Restrict, and Protect. The key difference between Restrict and Protect is that Protect is totally silent (no logs!).

  • A. off: (Incorrect) Not a valid violation mode.
  • B. protect: (Correct) Drops unauthorized frames but does NOT increment the violation counter and does NOT send syslog messages. It is completely silent.
  • C. restrict: (Incorrect) Drops unauthorized frames, but DOES send a notification/log to the admin.
  • D. shutdown: (Incorrect) Kills the entire port completely.
16. What security benefit is gained from enabling BPDU guard on PortFast enabled interfaces?
Correct Answer: preventing rogue switches from being added to the network

Deep Dive: PortFast speeds up ports facing end-users (like PCs) by skipping STP listening/learning phases. However, this is dangerous if a user plugs in a switch and creates a loop! A BPDU is a message that ONLY switches send. By enabling BPDU Guard, if that port ever receives a switch message, it instantly err-disables, stopping the rogue switch dead in its tracks.

  • A. preventing rogue switches: (Correct) If someone plugs a rogue switch into a wall jack, it sends BPDUs. BPDU guard sees this and kills the port.
  • B. protecting against Layer 2 loops: (Incorrect) Regular STP does this. BPDU guard is a specific defense mechanism for edge ports.
  • C. enforcing root bridges: (Incorrect) This is the job of 'Root Guard', not BPDU Guard.
  • D. buffer overflow: (Incorrect) Completely unrelated to STP.
17. Which type of VLAN-hopping attack may be prevented by designating an unused VLAN as the native VLAN?
Correct Answer: VLAN double-tagging

Deep Dive: A hacker can exploit the Native VLAN because traffic on it crosses Trunk links without a VLAN tag. If a hacker is assigned to the Native VLAN, they can wrap their packet in TWO tags (Double-Tagging). The first switch strips the outer tag (thinking it's native), and forwards the hidden inner tag to a second switch, allowing the hacker to hop VLANs. Moving the Native VLAN to an unused "black hole" number (like 999) completely stops this.

  • A. DTP spoofing: (Incorrect) This is prevented by turning off DTP (switchport nonegotiate), not by changing the Native VLAN.
  • C. VLAN double-tagging: (Correct) Changing the Native VLAN to an unused VLAN ID isolates the trunk's untagged traffic away from user PCs.
18. PC1 and PC2 should be able to obtain IP address assignments from the DHCP server. How many ports among the switches should be assigned as TRUSTED ports as part of the DHCP snooping configuration?
Correct Answer: 7

Deep Dive: When setting up DHCP Snooping, you must configure "Trust" on EVERY single port that connects backward toward the legitimate DHCP Server. (Note: Based on the original exhibit in the PDF, there is a core switch holding the DHCP server, and multiple access switches. You must trust the actual port connecting to the server [1], plus BOTH sides of every trunk link between the switches [2 trunks * 2 switches = 4], plus the uplinks [2], totaling 7 trusted ports facing upstream). User ports facing PCs are always left Untrusted.

19. An IT security specialist enables port security on a switch port. What is the default violation mode in use until the switch port is configured to use a different violation mode?
Correct Answer: shutdown

Deep Dive: When you type switchport port-security, three default things happen invisibly in the background: 1) Maximum allowed MACs is set to 1. 2) MAC addresses are NOT sticky. 3) The Violation Mode defaults to Shutdown. You only need to manually type a violation command if you want to change it to Restrict or Protect.

  • A. restrict: (Incorrect) Must be manually configured.
  • C. protect: (Incorrect) Must be manually configured.
  • D. shutdown: (Correct) The default action. It kills the port into an err-disabled state.
20. A network administrator enters: interface range fa0/5–10 followed by ip dhcp snooping limit rate 6. What is the effect?
Correct Answer: If any port receives more than 6 DHCP messages per second, the port will be shut down.

Deep Dive: To stop a DHCP Starvation attack (like Gobbler spamming discovery packets), DHCP Snooping allows you to rate-limit user ports. Standard Cisco IOS behavior dictates that if an untrusted port violates this rate limit (sends more than 6 packets per second), the switch assumes it's an attack and throws the port into an err-disabled (shutdown) state to protect the server.

  • B. Receive 6 of any type: (Incorrect phrasing) While it applies to all DHCP packets, the critical *effect* is what happens when the limit is breached.
  • C. Continue to operate: (Incorrect) The switch does not just log it; it violently shuts the port down for protection.
  • D. Port will be shut down: (Correct) The port enters err-disabled state upon breaching the limit.
21. A network administrator is configuring DAI on a switch with the command ip arp inspection validate src-mac. What is the purpose of this configuration command?
Correct Answer: It checks the source MAC address in the Ethernet header against the sender MAC address in the ARP body.

Deep Dive: Dynamic ARP Inspection (DAI) is an incredibly smart security feature. Hackers spoof ARP messages by putting a fake MAC address *inside* the payload of the ARP packet. The `validate src-mac` command forces the switch to open the packet and compare the Source MAC on the outside envelope (Ethernet header) to the Sender MAC written on the letter inside (ARP body). If they don't match, the hacker is lying, and the switch drops the packet!

  • A & B. MAC address table / ACLs: (Incorrect) This specific `validate` command does not check the MAC table or ACLs; it performs a cross-check within the packet itself.
  • C. target MAC address: (Incorrect) The target is who the packet is going TO. We are validating the sender (who the packet is coming FROM).
  • D. sender MAC address in the ARP body: (Correct) The switch verifies the outside Source MAC perfectly matches the inside Sender MAC.
22. Which two commands can be used to enable BPDU guard on a switch? (Combined)
Correct Answer: spanning-tree portfast bpduguard default (Global) AND spanning-tree bpduguard enable (Interface)

Deep Dive: BPDU Guard instantly err-disables a port if it hears a switch message (BPDU), which protects against users plugging in unauthorized switches. You can turn this on in two ways. 1) Globally, for EVERY port that has PortFast enabled. 2) Manually, on a specific interface.

  • A. Correct syntax: In global config mode, you type `spanning-tree portfast bpduguard default`. In interface config mode, you type `spanning-tree bpduguard enable`.
  • B & C. Incorrect syntax: The commands are spelled backward or missing the "portfast" keyword in the global command.
23. As part of a new security policy, all switches automatically learn MAC addresses for each port. All running configurations are saved daily. A thunderstorm causes a power outage. When the switches reboot, the dynamically learned MAC addresses are retained. Which port security configuration enabled this?
Correct Answer: sticky secure MAC addresses

Deep Dive: Normal dynamic Port Security learns MAC addresses and puts them in RAM. If the power dies, RAM is wiped clean! But if you use Sticky MAC, the switch learns the MAC address dynamically and instantly "pastes" it into the running configuration. When the admin saved the config at the end of the day, those Sticky MACs were saved to NVRAM permanently.

  • A. auto secure: (Incorrect) Not a real Cisco command.
  • B. dynamic secure: (Incorrect) Purely dynamic MACs vanish when the switch loses power.
  • C. static secure: (Incorrect) This means the admin manually typed the MAC address, but the question says they were "automatically learned".
  • D. sticky secure: (Correct) "Sticky" bridges the gap by automatically learning the MAC and saving it statically to the config file.
24. Which type of management frame may regularly be broadcast by an AP?
Correct Answer: beacon

Deep Dive: To help your smartphone find a Wi-Fi network, the Access Point operates in Passive Discover Mode. It continuously shouts out a Beacon frame multiple times a second, saying "Hi, I am Starbucks Wi-Fi, and here are my security settings!"

  • A. beacon: (Correct) The only management frame regularly broadcasted continuously by an AP.
  • B. probe request: (Incorrect) This is sent by your *client device* (phone/laptop) to ask if a specific network is nearby.
  • C. authentication: (Incorrect) Sent as a direct unicast during the login phase, not broadcasted regularly.
  • D. probe response: (Incorrect) This is sent by the AP, but only as a direct, one-time reply to a client's probe request.
25. What type of wireless antenna is best suited for providing coverage in large open spaces, such as hallways or large conference rooms?
Correct Answer: omnidirectional

Deep Dive: Antennas come in different shapes to push radio waves in different patterns. An Omnidirectional antenna pushes signal out in a 360-degree circle (like a donut), making it perfect for the ceiling of an office where laptops are scattered everywhere.

  • A. omnidirectional: (Correct) Best for 360-degree indoor coverage.
  • B, C, D. directional / Yagi / dish: (Incorrect) These antennas focus the radio wave into a tight, narrow laser beam to travel extremely long distances (like connecting two buildings across a street). They are terrible for an open conference room.
26. What is an advantage of SSID cloaking?
Correct Answer: Clients will have to manually identify the SSID to connect to the network.

Deep Dive: SSID Cloaking turns off the AP's Beacon broadcasts (Active Discover Mode). Because the name is hidden, casual users won't see it on their phones. They must manually type the exact network name to join.

  • B. Clients manually identify: (Correct) It adds a minor layer of privacy by hiding the name from average users.
  • C. Very difficult to discover: (Incorrect) Hackers can still discover hidden SSIDs in seconds using a packet sniffer to capture someone else's login request.
  • D. Best way to secure: (Incorrect) It is a very weak security feature. WPA2/WPA3 encryption is the *best* way to secure a network.
27. What are the two methods that are used by a wireless NIC to discover an AP? (Combined)
Correct Answer: transmitting a probe request AND receiving a broadcast beacon frame

Deep Dive: Your laptop's network card can find Wi-Fi in two ways: Passive Mode (sitting quietly and waiting to hear a Beacon frame from an AP) or Active Mode (shouting a Probe Request out to the world saying "Is the 'Guest_WiFi' network around here?").

  • D. Probe request & Beacon frame: (Correct) These represent the Active and Passive discovery mechanisms for 802.11 Wi-Fi.
  • A, B, C. ARP & 3-Way Handshake: (Incorrect) ARP is for finding MAC addresses on Ethernet. The 3-way handshake is for TCP connections. Neither are used for discovering radio waves.
28. Which wireless network topology would be used by network engineers to provide a wireless network for an entire college building?
Correct Answer: infrastructure

Deep Dive: When you connect multiple Access Points together via a wired backbone (so you can walk across a whole college campus without losing Wi-Fi), you are creating an Extended Service Set (ESS). This setup is formally known as Infrastructure Mode.

  • A. ad hoc: (Incorrect) This is peer-to-peer (like two laptops sending files to each other over Bluetooth/Wi-Fi Direct without a router).
  • B. hotspot: (Incorrect) A tiny, personal network running off cellular data (like your phone).
  • C. infrastructure: (Correct) The standard enterprise setup utilizing APs wired back to central switches.
29. What is a wireless security mode that requires a RADIUS server to authenticate wireless users?
Correct Answer: enterprise

Deep Dive: Large companies cannot use a single shared Wi-Fi password; if an employee gets fired, they'd have to change the password on every device in the building! Instead, they use WPA2-Enterprise (or WPA3-Enterprise), which links the Wi-Fi directly to a RADIUS Server. Every employee must log in with their own unique username and password.

  • A. enterprise: (Correct) Enterprise mode relies on an 802.1X/RADIUS server for centralized authentication.
  • B & C. personal / shared key: (Incorrect) This is WPA2-Personal, where everyone uses the exact same Pre-Shared Key (PSK).
  • D. WEP: (Incorrect) WEP is an incredibly old, broken encryption protocol.
30. What two IEEE 802.11 wireless standards operate ONLY in the 5 GHz range? (Combined)
Correct Answer: 802.11a and 802.11ac

Deep Dive: The IEEE 802.11 standards govern Wi-Fi frequencies. You must memorize these bands:

  • 802.11b & 802.11g: Operate ONLY on the 2.4 GHz band.
  • 802.11a & 802.11ac: Operate ONLY on the 5.0 GHz band. (Correct!)
  • 802.11n & 802.11ax: Dual-band. They operate on BOTH 2.4 GHz and 5.0 GHz simultaneously.
31. A technician is configuring the channel on a wireless router to either 1, 6, or 11. What is the purpose of adjusting the channel?
Correct Answer: to avoid interference from nearby wireless devices

Deep Dive: In the 2.4 GHz Wi-Fi band, there are 11 channels available (in North America). However, radio waves spread out and bleed into adjacent channels. Channel 2 bleeds into 1 and 3, causing massive static and lag. The only three channels that are spaced far enough apart to NEVER overlap each other are 1, 6, and 11.

  • A. disable broadcasting of the SSID: (Incorrect) This is called SSID Cloaking, not channel selection.
  • B. enable different 802.11 standards: (Incorrect) Standards (like 802.11n or 802.11ac) dictate the speed and frequency band used, not the specific channel within the band.
  • C. provide stronger security: (Incorrect) Security is handled by WPA2/WPA3, which operates independently of the radio frequency.
  • D. avoid interference: (Correct) Setting multiple APs exclusively to channels 1, 6, and 11 ensures their radio waves never crash into each other.
32. While attending a conference, participants are using laptops for network connectivity. When a guest speaker attempts to connect to the network, the laptop fails to display any available wireless networks. The access point must be operating in which mode?
Correct Answer: active

Deep Dive: In normal Wi-Fi networks (Passive mode), the AP shouts out "Beacon" frames so phones and laptops can see its name on the Wi-Fi list. If an AP stops sending Beacons to hide its name, it forces the client laptops into Active Mode. The client must actively send out a "Probe Request" asking for the network by its exact hidden name.

  • A. active: (Correct) The AP is hiding its SSID, forcing clients into Active mode (sending probe requests manually) to discover it.
  • B. mixed: (Incorrect) Mixed mode just means the AP supports older standards (like 802.11b) and newer standards (802.11n) at the same time.
  • C. open: (Incorrect) Open means there is no password required (like a coffee shop).
  • D. passive: (Incorrect) In passive mode, the AP openly broadcasts its name, which means the laptop would have seen it on the list.
33. A network administrator is required to upgrade wireless access to end users in a building. To provide data rates up to 1.3 Gb/s and still be backward compatible with older devices, which wireless standard should be implemented?
Correct Answer: 802.11ac

Deep Dive: You need to know the basic speed jumps of IEEE Wi-Fi standards. 802.11b and 802.11g are extremely old and slow (max 54 Mbps). 802.11n brought us up to 600 Mbps using MIMO antennas. 802.11ac is the first standard to breach the "Gigabit" speed barrier, providing 1.3 Gbps while remaining backward compatible.

  • A. 802.11n: (Incorrect) Caps out around 600 Mbps. It cannot reach 1.3 Gbps.
  • B. 802.11ac: (Correct) Operates in the 5GHz band, hits 1.3 Gbps, and allows backward compatibility with older devices.
  • C & D. 802.11g / 802.11b: (Incorrect) These are legacy 2.4 GHz standards that are incredibly slow.
34. A company has recently implemented an 802.11n wireless network. Some users are complaining that the wireless network is too slow. Which solution is the best method to enhance the performance of the wireless network?
Correct Answer: Split the traffic between the 2.4 GHz and 5 GHz frequency bands.

Deep Dive: The 802.11n standard is "Dual-Band," meaning it broadcasts both a 2.4 GHz and 5 GHz signal simultaneously. If the network is slow, it is usually because all the devices are piling onto the congested 2.4 GHz band. Renaming the network into two separate SSIDs (e.g., "WiFi" and "WiFi_5G") allows you to force laptops streaming video onto the faster 5 GHz band, clearing up traffic for everyone else.

  • A. Replace NICs: (Incorrect) Very expensive and doesn't fix a crowded radio frequency problem.
  • B. Split the traffic: (Correct) Pushing heavy users to the 5 GHz band heavily reduces congestion.
  • C. Disable DHCP: (Incorrect) DHCP has absolutely no impact on the speed of the wireless connection.
  • D. Upgrade firmware: (Incorrect) While firmware updates fix bugs, they generally do not magically boost network throughput on a congested band.
35. A technician is about to install and configure a wireless network at a small branch office. What is the first security measure the technician should apply immediately upon powering up the wireless router?
Correct Answer: Change the default user-name and password of the wireless router.

Deep Dive: Every home/small office router ships from the factory with a public, widely-known administrative login (like username: admin, password: admin). If you plug it in and configure Wi-Fi passwords first, a hacker could simply log into the router using the default admin password and lock you out! Your absolute first step is always securing the administrative login.

  • C. Change the default user-name and password: (Correct) This protects the device's configuration portal from being hijacked immediately.
  • A, B, D. Encryption / SSID Cloaking / MAC filtering: (Incorrect) These are all great Wi-Fi security steps to take, but they must be done after you have secured the router's main administrative login portal.
36. On a Cisco 3504 WLC dashboard, which option provides access to the full menu of features?
Correct Answer: Advanced

Deep Dive: When you log into a Cisco WLC (Wireless LAN Controller), you are greeted with a simplified dashboard (The Network Summary page) showing pie charts of clients and APs. To actually configure complex settings like RADIUS servers, VLAN interfaces, and DHCP scopes, you must click the "Advanced" button in the top right corner.

  • B. Advanced: (Correct) Clicking this leaves the basic dashboard and opens the full, professional GUI menu.
  • A, C, D. Rogues / APs / Network Summary: (Incorrect) These are just informational sections on the basic landing page dashboard.
37. On a Cisco 3504 WLC Summary page (Advanced > Summary), which tab allows a network administrator to access and configure a WLAN for a specific security option such as WPA2?
Correct Answer: WLANs

Deep Dive: This is a slightly tricky GUI question. While there is a "SECURITY" tab, that tab is used globally to add RADIUS servers to the WLC. If you want to apply WPA2 security to a specific Wi-Fi network (WLAN), you must click the WLANs tab, edit the specific network profile, and apply the security settings to it there.

  • A. WLANs: (Correct) You click here to create network profiles, assign them to interfaces, and configure their specific WPA2/WPA3 passwords.
  • B. SECURITY: (Incorrect) Used to add global AAA/RADIUS servers, not to assign WPA2 to a specific Wi-Fi network.
  • C. WIRELESS: (Incorrect) Used to view and manage physical Access Points.
  • D. MANAGEMENT: (Incorrect) Used to set up SNMP traps and device administration.
38. Which protocol can be used to monitor the network?
Correct Answer: SNMP

Deep Dive: Network administrators can't manually check 500 switches every day to see if they are overheating or if ports are failing. They use SNMP (Simple Network Management Protocol). The WLC and switches automatically send "Traps" (log messages/alerts) to a central SNMP server whenever something goes wrong.

  • B. SNMP: (Correct) The industry standard for monitoring network device health and performance.
  • A & D. AAA / RADIUS: (Incorrect) These are used for security, access control, and user logins, not health monitoring.
  • C. DHCP: (Incorrect) Used to hand out IP addresses.
39. A network administrator deploys a wireless router in a small law firm. Employee laptops join the WLAN and receive IP addresses in the 10.0.10.0/24 network. Which service is used on the wireless router to allow the employee laptops to access the internet?
Correct Answer: NAT

Deep Dive: The 10.0.10.0/24 network is a Private IP block (RFC 1918). Private IPs are strictly banned from the public internet. If a laptop tries to go to Google with a 10.x.x.x address, the ISP drops it instantly. The wireless router uses NAT (Network Address Translation) to erase the private IP on the packet and replace it with the router's single, public IP address before sending it to the internet.

  • A. DNS: (Incorrect) Turns names (google.com) into IPs. Does not route traffic.
  • B. NAT: (Correct) Translates private unroutable IPs into public routable IPs.
  • C. DHCP: (Incorrect) Hands out the 10.0.10.0 addresses, but does not allow them to cross the internet boundary.
  • D. RADIUS: (Incorrect) Used for user authentication.
40. Which service can be used on a wireless router to prioritize network traffic among different types of applications so that voice and video data are prioritized over email and web data?
Correct Answer: QoS

Deep Dive: If you are downloading a massive 50GB file, it might clog the router, causing your zoom video calls to freeze. We fix this by enabling QoS (Quality of Service). QoS allows you to tell the router: "Voice and Video packets are VIPs; send them out instantly. File downloads can wait in line."

  • A. NAT: (Incorrect) Translates private IPs to public IPs.
  • B. QoS: (Correct) Inspects packets (often using the DSCP field) and prioritizes time-sensitive traffic over bulk data.
  • C & D. DNS / DHCP: (Incorrect) Completely unrelated to traffic speed and prioritization.
41. Which step is required before creating a new WLAN on a Cisco 3500 series WLC?
Correct Answer: Create a new VLAN interface.

Deep Dive: On a WLC, every single Wi-Fi network (WLAN) you broadcast needs to be mapped to a specific wired VLAN so the traffic knows where to go once it hits the physical switch. Therefore, before you can create a WLAN profile, you MUST go to the "Interfaces" tab and create a virtual interface bound to a specific VLAN ID.

  • A. Create a new SSID: (Incorrect) You create the SSID during the creation of the WLAN, not before.
  • B. Create a new VLAN interface: (Correct) The WLC needs a Virtual Interface to tie the wireless traffic to the wired network.
  • C & D. SNMP / RADIUS: (Incorrect) These are optional management and security features. You can create an open or WPA2-Personal WLAN without either of these servers.
42. A network engineer is troubleshooting a newly deployed wireless network. When users access streaming video, the performance is poor. To improve performance, the engineer configures a 5 GHz SSID and trains users to use it for streaming. Why might this solution improve performance?
Correct Answer: The 5 GHz band has more channels and is less crowded than the 2.4 GHz band...

Deep Dive: The 2.4 GHz band is extremely congested. It only has 3 non-overlapping channels (1, 6, and 11) and it shares airspace with microwaves, baby monitors, and Bluetooth. The 5 GHz band has 24 non-overlapping channels and operates much faster, making it perfect for heavy video streaming.

  • A. 5 GHz has greater range: (Incorrect) 5 GHz actually has a shorter range than 2.4 GHz because higher frequencies cannot penetrate walls as easily.
  • B & D. Inconvenient / limits users: (Incorrect) The goal of a network engineer is to improve the network, not actively make it inconvenient so people stop using it!
  • C. More channels and less crowded: (Correct) Moving heavy bandwidth users to the wider, less congested 5 GHz band frees up the network for everyone.
43. A network administrator is working to improve WLAN performance on a dual-band wireless router. What is a simple way to achieve a split-the-traffic result?
Correct Answer: Make sure that different SSIDs are used for the 2.4 GHz and 5 GHz bands.

Deep Dive: By default, many dual-band routers broadcast the exact same SSID for both 2.4 GHz and 5 GHz bands. Because 2.4 GHz travels further, laptops often connect to it by mistake, causing network congestion. The easiest fix is to name them differently (e.g., "OfficeNet" and "OfficeNet_5G") so users can deliberately choose the faster band.

  • A. Require 802.11n: (Incorrect) This might actually block older devices from connecting completely.
  • B. Keep firmware updated: (Incorrect) Good practice, but it does not "split" traffic.
  • C. Different SSIDs: (Correct) This forces physical separation of traffic based on user choice.
  • D. Add a range extender: (Incorrect) Extenders generally cut bandwidth in half because they have to repeat the signal. It would make performance worse.
44. A network administrator is configuring a RADIUS server connection on a Cisco WLC. The configuration requires a shared secret password. What is the purpose of the shared secret password?
Correct Answer: It is used to encrypt the messages between the WLC and the RADIUS server.

Deep Dive: In an Enterprise environment, the WLC has to send usernames and passwords over the network to the RADIUS server to verify them. If a hacker is listening, they could steal the passwords! To stop this, the WLC and RADIUS server share a "Secret Key" that they use to encrypt their conversation. It has nothing to do with the end user's Wi-Fi password.

  • A, B, C. User authentication/encryption: (Incorrect) The end user never sees or types this shared secret. The user types their own personal username/password to get on the Wi-Fi.
  • D. Encrypt messages between WLC and RADIUS: (Correct) The shared secret secures the backend communication channel between the network infrastructure devices.
45. A laptop cannot connect to a wireless access point. Which two troubleshooting steps should be taken first? (Combined)
Correct Answer: Ensure that the wireless NIC is enabled AND Ensure that the wireless SSID is chosen.

Deep Dive: Troubleshooting always follows the OSI model from the bottom up. Start with the absolute basics (Layer 1 / Layer 2). Is the laptop's Wi-Fi card actually turned on? (Many laptops have physical switches or Airplane mode). Once it's on, did the user actually select the correct SSID from the list?

  • A. NIC enabled & SSID chosen: (Correct) These are the most common, fundamental Layer 1 and Layer 2 issues for end-users.
  • B & C. Antenna attached / frequency configured: (Incorrect) Laptops have built-in internal antennas, and frequency selection is handled automatically by the NIC and AP negotiating.
46. Which three parameters would need to be changed if best practices are being implemented for a brand new home wireless AP? (Combined)
Correct Answer: SSID, AP password, wireless network password

Deep Dive: When you buy a brand new router from a store, it has a default SSID (like "Netgear5G"), a default Wi-Fi password, and a widely-known default Admin password. If you leave these, anyone can Google your router's default login and hijack it. You must change all three immediately.

  • A & C. Frequency / Beacon time / Client OS password: (Incorrect) Beacon intervals don't need to be changed for security. Antenna frequency is auto-negotiated. The router has no control over the user's Windows/Mac OS password.
  • B. SSID, AP password, wireless network password: (Correct) The golden triad of basic home router security.
47. Which access control component, implementation, or protocol controls what users can DO on the network?
Correct Answer: authorization

Deep Dive: AAA stands for Authentication, Authorization, and Accounting.

  • Authentication: Proves WHO you are.
  • Authorization: (Correct) Dictates WHAT you can DO (e.g., Are you allowed to access the HR server, or just the guest internet?).
  • Accounting: Tracks WHAT you DID.
  • 802.1X: Is a port-based authentication protocol, not the component that determines authorization levels.
48. Which access control component, implementation, or protocol is implemented either locally or as a server-based solution?
Correct Answer: authentication

Deep Dive: You can verify a user's identity (Authentication) in two ways. You can use Local authentication, where you type the username/password directly into the router's local database. Or, you can use a Server-based solution, where the router forwards the credentials to a centralized RADIUS server to check the database there.

49. Which access control component, implementation, or protocol AUDITS what user actions are performed on the network?
Correct Answer: accounting

Deep Dive: The third "A" in AAA stands for Accounting. It keeps a strict, timestamped logbook. It audits and tracks exactly what time you logged in, how much data you transferred, and what commands you executed. It is crucial for forensic investigations.

50. Which access control component, implementation, or protocol restricts LAN access through publicly accessible switch ports?
Correct Answer: 802.1X

Deep Dive: If your company has a waiting room with an Ethernet wall jack, anyone could plug a laptop in and hack the network! To stop this, we use 802.1X. It is an IEEE protocol that keeps the physical switch port mathematically disconnected until the user's PC provides a valid username and password (which is then verified by an Authentication Server).

51. Which access control component, implementation, or protocol logs EXEC and configuration commands configured by a user?
Correct Answer: accounting

Deep Dive: In the AAA framework, keeping track of every single command a user types (like `enable` or `configure terminal`) is crucial for auditing. If someone accidentally deletes a router configuration, you need to know exactly who did it and when.

  • A. authorization: (Incorrect) Controls if you are allowed to type the command, but doesn't record it.
  • B. accounting: (Correct) The logging and tracking mechanism. It audits EXEC session commands and configuration changes.
  • C. authentication: (Incorrect) Verifies identity, not actions.
  • D. 802.1X: (Incorrect) Controls physical port access.
52. Which access control component, implementation, or protocol controls WHO is permitted to access a network?
Correct Answer: authentication

Deep Dive: Be careful with the wording here! Authorization controls WHAT you can do, but Authentication acts as the front door. It controls WHO is permitted to enter the network in the first place by checking usernames and passwords.

  • A. authorization: (Incorrect) Decides permissions after you are already in.
  • B. accounting: (Incorrect) Keeps a log of what you did.
  • C. authentication: (Correct) The exact process of verifying a user's identity to permit access.
  • D. 802.1X: (Incorrect) While it *uses* authentication to open a port, the core component defining "who" is permitted is Authentication itself.
53. Which access control component, implementation, or protocol collects and reports usage data?
Correct Answer: accounting

Deep Dive: If your ISP bills you for how much data you download, they use Accounting to track it. In a corporate network, AAA Accounting collects usage data (session start/stop times, bytes transferred) for auditing and billing purposes.

  • A. authorization: (Incorrect) Sets permissions.
  • B. accounting: (Correct) Collects, logs, and reports network usage data.
  • C. authentication: (Incorrect) Verifies identity.
  • D. 802.1X: (Incorrect) Only handles initial access to the switch port.
54. Which access control component, implementation, or protocol indicates success or failure of a client-requested service with a PASS or FAIL message?
Correct Answer: authentication

Deep Dive: When you type in a password, the system checks the database. It immediately returns a simple binary response: PASS (the credentials match) or FAIL (the credentials do not match). This direct feedback is the core of Authentication.

  • A. authorization: (Incorrect) Authorization silently applies a list of rules; it doesn't give you a binary PASS/FAIL login message.
  • B. accounting: (Incorrect) Silent background logging.
  • C. authentication: (Correct) The exact process that verifies credentials and returns a PASS or FAIL access message.
55. Which access control component, implementation, or protocol is based on device roles of supplicant, authenticator, and authentication server?
Correct Answer: 802.1X

Deep Dive: This specific triad of roles (Supplicant, Authenticator, Authentication Server) is the exact, defining architecture of 802.1X. The PC (Supplicant) asks the Switch (Authenticator) to open the port. The Switch asks the RADIUS Server (Auth Server) if the password is correct.

  • A, B, C. AAA Components: (Incorrect) These are general concepts. Only 802.1X specifically defines these three physical/logical device roles.
  • D. 802.1X: (Correct) The IEEE standard for port-based Network Access Control (PNAC).
56. Which access control component, implementation, or protocol is based upon usernames and passwords?
Correct Answer: authentication

Deep Dive: Usernames, passwords, biometrics, and security tokens are all forms of credentials. The process of verifying those credentials to prove your identity is Authentication.

  • A & B. AuthZ / Accounting: (Incorrect) These do not use passwords. They use the identity established by the authentication phase.
  • C. authentication: (Correct) The fundamental process relying on usernames and passwords.
  • D. 802.1X: (Incorrect) 802.1X uses authentication, but it is a broader port-security protocol.
57. Which type of wireless network uses transmitters to provide coverage over an extensive geographic area?
Correct Answer: wireless wide-area network (WWAN)

Deep Dive: An "extensive geographic area" means entire countries, regions, or even global coverage. A WWAN uses massive infrastructure like 4G/5G cellular towers and satellites to achieve this.

  • A. WLAN: (Incorrect) Only covers a single home or building (Wi-Fi).
  • B. WMAN: (Incorrect) Covers a single city or district.
  • C. WPAN: (Incorrect) Only covers a few feet around your body.

58. Which type of wireless network commonly uses Bluetooth or ZigBee devices?
Correct Answer: wireless personal-area network (WPAN)

  • A, B, D. WLAN / WMAN / WWAN: (Incorrect) These networks cover much larger areas using much higher-powered transmitters (like Wi-Fi routers or cell towers).
  • C. WPAN: (Correct) Specifically designed for short-range personal devices.
59. Which type of wireless network uses transmitters to provide wireless service over a large urban region?
Correct Answer: wireless metropolitan-area network (WMAN)

  • B. WMAN: (Correct) Specifically designed for urban regions and cities.

60. Which type of wireless network is suitable for use in a home or office?
Correct Answer: wireless local-area network (WLAN)

Deep Dive: A Local Area Network (LAN) is confined to a single geographic location, like a house, school, or office building. When you add Wi-Fi to it, it becomes a WLAN. This provides coverage up to about 300 feet, which is perfect for laptops and phones moving around a house.

  • A. WLAN: (Correct) The standard implementation of Wi-Fi in homes and offices.
  • B & D. WMAN / WWAN: (Incorrect) Used for massive outdoor city/national coverage.
  • C. WPAN: (Incorrect) Too small to cover a whole house (Bluetooth drops after a few rooms).
61. Which type of wireless network often makes use of devices mounted on buildings?
Correct Answer: wireless metropolitan-area network

Deep Dive: To cover an entire city or district, ISPs mount large, high-powered wireless transmitters on the roofs of tall buildings or towers. This specific city-wide setup is a WMAN.

  • A & D. WLAN / WPAN: (Incorrect) These are used inside single homes or rooms.
  • B. WMAN: (Correct) Specifically designed to bounce signals off building-mounted receivers to blanket a city.

62. Which type of wireless network is suitable for national and global communications?

63. Which type of wireless network uses transmitters to cover a medium-sized network, usually up to 300 feet (91.4 meters)?

67. Match each functional component of AAA with its correct description. (Combined)
Correct Answer: Authentication = Proves identity | Authorization = Determines access | Accounting = Records actions

Deep Dive: This is the golden rule of AAA. Authentication is the username/password check (Who are you?). Authorization applies the rules (Are you allowed in the server room?). Accounting is the history logbook (You entered the server room at 3:00 PM).

68. Which two Cisco solutions help prevent DHCP starvation attacks? (Choose two, Combined)
Correct Answer: Port Security AND DHCP Snooping

Deep Dive: DHCP Starvation happens when a hacker's tool (like Gobbler) constantly changes its MAC address to steal every IP in the DHCP pool. You can stop this in two ways: 1) Port Security physically limits the port to 1 or 2 MAC addresses, instantly shutting down the hacker's tool. 2) DHCP Snooping allows you to set a "limit rate" on user ports (e.g., max 5 DHCP packets per second), stopping the flood.

69. What are three techniques for mitigating VLAN attacks? (Choose three, Combined)
Correct Answer: Disable DTP, Enable trunking manually, Set the native VLAN to an unused VLAN.

Deep Dive: To stop VLAN Hopping, you must lock down your trunks. 1) Disable DTP auto-negotiation so hackers can't trick the switch into forming a trunk. 2) Manually configure legitimate trunks (switchport mode trunk). 3) Change the Native VLAN to a dummy/unused number (like VLAN 999) to stop Double-Tagging attacks.

70. Refer to an exhibit showing show port-security interface with the output "Violation Mode: Shutdown". What can be determined about port security from this information?
Correct Answer: The port violation mode is the default for any port that has port security enabled.

Deep Dive: When you enable Port Security, the Violation Mode defaults to Shutdown. You only see "Restrict" or "Protect" if an administrator manually configures it.

  • A. violation mode is the default: (Correct) Shutdown is the factory default behavior.
  • B. port has been shut down: (Incorrect) The status shows "Secure-up", meaning it is currently active, it just *will* shut down if a violation occurs.
71. A network administrator of a college is configuring the WLAN user authentication process. Wireless users are required to enter username and password credentials that will be verified by a server. Which server would provide such service?
Correct Answer: RADIUS

Deep Dive: WPA2-Enterprise Wi-Fi requires each user to have their own unique username and password. The WLC forwards these credentials to an external RADIUS Server to verify them against a database (like Active Directory).

  • A. AAA: (Incorrect) AAA is the general *concept/framework*, but RADIUS is the actual *server/protocol* used.
  • B. NAT: (Incorrect) Translates IP addresses.
  • C. SNMP: (Incorrect) SNMP is used for monitoring device health.
  • D. RADIUS: (Correct) Remote Authentication Dial-In User Service is the server software that authenticates Wi-Fi users.
72. A technician is troubleshooting a slow WLAN consisting of 802.11b/g devices. A new 802.11n/ac dual-band router replaces the old one, but streaming performance is still poor. What can the technician do to address the slow wireless speed?
Correct Answer: Split the wireless traffic between the 802.11n 2.4 GHz band and the 5 GHz band.

Deep Dive: Because 802.11ac routers are dual-band, they broadcast both 2.4 GHz and 5 GHz. If the network is slow, it means all the older devices and new devices are piling into the crowded 2.4 GHz frequency. By giving the 5 GHz band a separate SSID (splitting the traffic), you can force high-bandwidth users (video streaming) onto the fast 5 GHz lane, leaving the 2.4 GHz lane clear for older devices.

73. The company handbook states that employees cannot have microwave ovens in their offices. Instead, all employees must use the microwaves in the cafeteria. What wireless security risk is the company trying to avoid?
Correct Answer: accidental interference

Deep Dive: Standard Wi-Fi (802.11b/g/n) operates heavily on the 2.4 GHz frequency band. The magnetron inside a standard microwave oven *also* operates at exactly 2.4 GHz! When someone turns on a cheap microwave, it blasts "noise" into the airwaves, causing massive accidental interference that will immediately disconnect nearby laptops from the WLAN.

74. What is the function provided by CAPWAP protocol in a corporate wireless network?
Correct Answer: CAPWAP provides the encapsulation and forwarding of wireless user traffic between an access point and a wireless LAN controller.

Deep Dive: CAPWAP is the "management tunnel" between a WLC and an Access Point. It has two jobs: 1) A Control channel (encrypted with DTLS) to push configuration settings to the AP. 2) A Data channel to encapsulate the end-user's internet traffic and tunnel it all the way back to the WLC for processing.

  • A. Encapsulation between AP and WLC: (Correct) This perfectly defines the CAPWAP Data tunnel.
  • B. IPv6 to IPv4: (Incorrect) CAPWAP has nothing to do with IP translation.
  • C. Encryption between AP and client: (Incorrect) Client encryption is handled by WPA2/AES over the radio waves, not CAPWAP.
75. Packet Tracer Scenario: You view the show port-security interface fa0/1 output and notice the line Violation Mode : Protect. Which event will take place if there is a port security violation on switch S1 interface Fa0/1?

  • A & B. Notifications/Syslog: (Incorrect) This describes the "Restrict" mode.
  • C. Packets dropped (silently): (Correct) This perfectly describes the "Protect" mode.
